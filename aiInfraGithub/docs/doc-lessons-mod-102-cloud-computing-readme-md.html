Module 02: 云计算（Cloud Computing） for 机器学习基础设施（ML Infrastructure）

Duration: 50 hours Difficulty: Intermediate Prerequisites: Module 01 complete

Module Overview

This module provides an in-depth exploration of cloud computing platforms for ML infrastructure. You'll learn to architect, deploy, and manage ML systems across AWS 云（AWS）, GCP 云（GCP）, and Azure 云（Azure）, with a focus on 成本优化（cost optimization）, scalability, and production best practices.

Learning Objectives

By the end of this module, you will be able to:

  1. Compare and choose appropriate cloud platforms for ML workloads
  2. Design cloud-native ML architectures across AWS 云（AWS）, GCP 云（GCP）, and Azure 云（Azure）
  3. Optimize costs for ML infrastructure in the cloud
  4. Deploy and manage compute resources (VMs, containers, serverless)
  5. Configure storage solutions for datasets and models
  6. Implement networking for secure and performant ML systems
  7. Use managed ML services (SageMaker, Vertex AI 托管 ML（Vertex AI）, Azure 云（Azure） ML)
  8. Build multi-cloud and hybrid strategies

Module Structure

Lesson 01: Cloud Architecture for ML (6 hours)
  • Cloud architecture patterns
  • Compute, storage, networking fundamentals
  • ML-specific considerations
  • Cost modeling and estimation
  • Hands-on: Design ML system architecture
Lesson 02: AWS 云（AWS） for 机器学习基础设施（ML Infrastructure） (7 hours)
  • AWS 云（AWS） ecosystem overview
  • EC2, S3, EBS, VPC
  • EKS 托管 Kubernetes（EKS） (Elastic Kubernetes 容器编排（Kubernetes） Service)
  • SageMaker 托管 ML（SageMaker） deep dive
  • IAM and security
  • Hands-on: Deploy ML system on AWS 云（AWS）
Lesson 03: Google Cloud Platform for ML (7 hours)
  • GCP 云（GCP） ecosystem overview
  • Compute Engine, Cloud Storage, GKE 托管 Kubernetes（GKE）
  • Vertex AI 托管 ML（Vertex AI） platform
  • TPU access and usage
  • GCP 云（GCP） networking
  • Hands-on: Deploy ML system on GCP 云（GCP）
Lesson 04: Azure 云（Azure） for 机器学习基础设施（ML Infrastructure） (6 hours)
  • Azure 云（Azure） ecosystem overview
  • Virtual Machines, Blob Storage, AKS 托管 Kubernetes（AKS）
  • Azure 云（Azure） 机器学习（Machine Learning）
  • Azure 云（Azure） OpenAI integration
  • Enterprise features
  • Hands-on: Deploy ML system on Azure 云（Azure）
Lesson 05: Cloud Storage for ML (6 hours)
  • Object storage (S3, GCS, Blob)
  • Block storage (EBS, Persistent Disks)
  • File storage (EFS, Filestore)
  • Data lakes and data warehouses
  • Caching strategies
  • Hands-on: Optimize data pipeline
Lesson 06: Cloud Networking for ML (6 hours)
  • VPC and subnet design
  • Load balancers and traffic management
  • CDN for model serving
  • VPN and hybrid connectivity
  • Service mesh (Istio)
  • Hands-on: Secure multi-tier ML system
Lesson 07: Managed ML Services (6 hours)
  • SageMaker 托管 ML（SageMaker） vs Vertex AI 托管 ML（Vertex AI） vs Azure 云（Azure） ML
  • 训练（Training） jobs and endpoints
  • Feature stores
  • Model registry and versioning
  • AutoML capabilities
  • Hands-on: Compare managed services
Lesson 08: Multi-Cloud and Cost Optimization (6 hours)
  • Multi-cloud strategies
  • Cloud cost management
  • Reserved instances and savings plans
  • Spot instances for training
  • Cost monitoring and alerts
  • Hands-on: Optimize cloud spending
