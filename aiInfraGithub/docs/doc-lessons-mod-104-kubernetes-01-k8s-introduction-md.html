Kubernetes å®¹å™¨ç¼–æŽ’ï¼ˆKubernetesï¼‰ (K8s) is an open-source container orchestration platform that automates deployment, scaling, and management of containerized applications.

Etymology

The name comes from Greek ÎºÏ…Î²ÎµÏÎ½Î®Ï„Î·Ï‚ (kubernetes), meaning "helmsman" or "pilot." The "K8s" abbreviation replaces the 8 letters "ubernete" with "8".

Created by Google, Now CNCF

Core Capabilities

Kubernetes å®¹å™¨ç¼–æŽ’ï¼ˆKubernetesï¼‰ provides:

  5. Automated rollouts and rollbacks - Safe deployments with zero downtime
  6. Secret and configuration management - Secure handling of sensitive data
  7. Resource optimization - Efficient bin-packing of workloads

Step 1: Create a Simple Flask ML API

app.py:

pythonPYTHON
from flask import Flask, request, jsonify
import numpy as np

app = Flask(__name__)

@app.route('/predict', methods=['POST'])
def predict():
    data = request.json
    # Simulate ML prediction
    input_data = np.array(data['features'])
    prediction = float(np.sum(input_data))  # Dummy model

    return jsonify({
        'prediction': prediction,
        'model_version': 'v1.0'
    })

@app.route('/health', methods=['GET'])
def health():
    return jsonify({'status': 'healthy'})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
