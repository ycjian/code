Question 11

Q: Which storage type provides the best performance for high-throughput ML training data access?

Answer: B) Block storage (EBS/Persistent Disk)

Explanation: Block storage delivers the highest IOPS and lowest per-request latency of the managed options, which matters for tight training loops that re-read shards quickly.
