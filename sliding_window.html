3. Longest Substring Without Repeating Characters

Difficulty: Medium | Key Concept: sliding window, hashmap

Original Problem

Given a string s, find the length of the longest substring without repeating characters.

中文翻译

给定字符串 s，找出其中不包含重复字符的最长连续子串，并返回该子串的长度。

Example

Input: s = "abcabcbb"

Output: 3 (Explanation: "abc")

Java
class Solution {
  public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int left = 0, maxLen = 0;
    for (int right = 0; right < s.length(); right++) {
      char c = s.charAt(right);
      if (map.containsKey(c) && map.get(c) >= left) {
        left = map.get(c) + 1;
      }
      map.put(c, right);
      maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
  }
}

Explanation / 解题说明

  • 使用滑动窗口，left和right指针维护无重复子串区间
  • HashMap记录每个字符最后一次出现的位置，遇到重复时移动left到重复字符后
  • 每次迭代更新最大长度

30. Substring with Concatenation of All Words

Difficulty: Hard | Key Concept: sliding window, hashmap

Original Problem

Given string s and array words (all same length), find all starting indices where a concatenation of all words (any order) appears as a substring.

中文翻译

给定字符串 s 和等长单词数组 words，找出所有起始下标，使得从该位置开始的子串恰好由 words 中全部单词按任意顺序拼接而成。

Example

Input: s = "barfoothefoobarman", words = ["foo","bar"]

Output: [0,9]

Java
class Solution {
  public List<Integer> findSubstring(String s, String[] words) {
    List<Integer> result = new ArrayList<>();
    if (words.length == 0) return result;
    int wordLen = words[0].length(), totalLen = wordLen * words.length;
    Map<String, Integer> need = new HashMap<>();
    for (String w : words) need.put(w, need.getOrDefault(w, 0) + 1);
    for (int i = 0; i < wordLen; i++) {
      int left = i, matched = 0;
      Map<String, Integer> window = new HashMap<>();
      for (int right = i; right + wordLen <= s.length(); right += wordLen) {
        String word = s.substring(right, right + wordLen);
        if (need.containsKey(word)) {
          window.put(word, window.getOrDefault(word, 0) + 1);
          if (window.get(word).equals(need.get(word))) matched++;
        }
        if (right - left + wordLen > totalLen) {
          String leftWord = s.substring(left, left + wordLen);
          if (need.containsKey(leftWord)) {
            if (window.get(leftWord).equals(need.get(leftWord))) matched--;
            window.put(leftWord, window.get(leftWord) - 1);
          }
          left += wordLen;
        }
        if (matched == need.size()) result.add(left);
      }
    }
    return result;
  }
}

Explanation / 解题说明

  • 单词长度固定，按单词分块滑动，从0到wordLen-1个起点开始
  • 使用两个HashMap记录需要和窗口内单词出现次数
  • 窗口长度超过总长度时移除左端单词

76. Minimum Window Substring

Difficulty: Hard | Key Concept: sliding window, two pointers, hashmap

Original Problem

Given two strings s and t, return the minimum window substring of s that contains all characters of t (including duplicates).

中文翻译

给定字符串 s 和 t，返回 s 中能够覆盖 t 全部字符的最短连续子串。t 中的重复字符也必须按出现次数覆盖。

Example

Input: s = "ADOBECODEBANC", t = "ABC"

Output: "BANC"

Java
class Solution {
  public String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>();
    for (char c : t.toCharArray()) need.put(c, need.getOrDefault(c, 0) + 1);
    int left = 0, matched = 0, minLen = Integer.MAX_VALUE, start = 0;
    for (int right = 0; right < s.length(); right++) {
      char c = s.charAt(right);
      if (need.containsKey(c)) {
        need.put(c, need.get(c) - 1);
        if (need.get(c) == 0) matched++;
      }
      while (matched == need.size()) {
        if (right - left + 1 < minLen) {
          minLen = right - left + 1;
          start = left;
        }
        char leftChar = s.charAt(left);
        if (need.containsKey(leftChar)) {
          need.put(leftChar, need.get(leftChar) + 1);
          if (need.get(leftChar) > 0) matched--;
        }
        left++;
      }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
  }
}

Explanation / 解题说明

  • need存储所需字符计数，matched记录完全满足的字符种类数
  • 右指针扩展直到包含全部所需字符，左指针收缩找最小窗口
  • 时间复杂度O(m+n)，空间复杂度O(k)

187. Repeated DNA Sequences

Difficulty: Medium | Key Concept: sliding window, HashSet

Original Problem

Given a string s that represents a DNA sequence, return all 10-letter-long substrings that occur more than once.

中文翻译

给定表示 DNA 序列的字符串 s，返回所有出现次数超过一次的长度为 10 的子串。答案中的重复子串只需要返回一次。

Example

Input: s = "AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT"

Output: ["AAAAACCCCC","CCCCCAAAAA"]

Java
class Solution {
  public List<String> findRepeatedDnaSequences(String s) {
    Set<String> seen = new HashSet<>();
    Set<String> repeated = new HashSet<>();

    for (int i = 0; i + 10 <= s.length(); i++) {
      // 固定截取长度为 10 的窗口。
      String substring = s.substring(i, i + 10);
      if (!seen.add(substring)) {
        repeated.add(substring);
      }
    }
    return new ArrayList<>(repeated);
  }
}

Explanation / 解题说明

  • 窗口长度固定为 10，依次提取所有候选 DNA 子串。
  • seen 记录已经出现过的子串，repeated 自动去重重复答案。
  • 时间复杂度为 O(n)，空间复杂度为 O(n)；由于窗口长度恒定为 10，截取成本视为常数。

209. Minimum Size Subarray Sum

Difficulty: Medium | Key Concept: sliding window, two pointers

Original Problem

Given an array of positive integers and a target sum, find the minimal length of a contiguous subarray of which the sum ≥ target. Return 0 if none.

中文翻译

给定一个正整数数组和目标值 target，找出元素和大于等于 target 的最短连续子数组长度。如果不存在满足条件的子数组，返回 0。

Example

Input: target = 7, nums = [2,3,1,2,4,3]

Output: 2 (Explanation: [4,3])

Java
class Solution {
  public int minSubArrayLen(int target, int[] nums) {
    int left = 0, sum = 0, minLen = Integer.MAX_VALUE;
    for (int right = 0; right < nums.length; right++) {
      sum += nums[right];
      while (sum >= target) {
        minLen = Math.min(minLen, right - left + 1);
        sum -= nums[left++];
      }
    }
    return minLen == Integer.MAX_VALUE ? 0 : minLen;
  }
}

Explanation / 解题说明

  • 右指针扩展窗口并累加和，当和满足条件时记录长度
  • 不断移动左指针缩小窗口直到不满足条件，寻找最小长度
  • 时间复杂度O(n)，每个元素最多访问两次

219. Contains Duplicate II

Difficulty: Easy | Key Concept: sliding window, HashSet

Original Problem

Given array nums and integer k, return true if there exist two distinct indices i and j such that nums[i] == nums[j] and |i - j| <= k.

中文翻译

给定数组 nums 和整数 k，判断是否存在两个不同下标 i 和 j，使得 nums[i] == nums[j]，并且两个下标之间的距离不超过 k。

Example

Input: nums = [1,2,3,1], k = 3

Output: true

Java
class Solution {
  public boolean containsNearbyDuplicate(int[] nums, int k) {
    Set<Integer> window = new HashSet<>();

    for (int i = 0; i < nums.length; i++) {
      // 加入前先检查，window 中只保留距离当前下标不超过 k 的元素。
      if (window.contains(nums[i])) {
        return true;
      }
      window.add(nums[i]);

      if (window.size() > k) {
        window.remove(nums[i - k]);
      }
    }
    return false;
  }
}

Explanation / 解题说明

  • 使用 HashSet 维护最近 k 个元素。
  • 新元素加入前，如果集合中已经存在相同值，就找到了满足距离限制的一对下标。
  • 时间复杂度为 O(n)，空间复杂度为 O(k)。

220. Contains Duplicate III

Difficulty: Hard | Key Concept: sliding window, TreeSet

Original Problem

Given array nums, integers indexDiff and valueDiff, return true if there exist distinct indices i and j such that |i - j| <= indexDiff and |nums[i] - nums[j]| <= valueDiff.

中文翻译

给定数组 nums、下标差上限 indexDiff 和数值差上限 valueDiff，判断是否存在两个不同下标，使它们的下标距离和对应数值之差都不超过给定上限。

Example

Input: nums = [1,2,3,1], indexDiff = 3, valueDiff = 0

Output: true

Java
class Solution {
  public boolean containsNearbyAlmostDuplicate(
      int[] nums, int indexDiff, int valueDiff) {
    TreeSet<Long> window = new TreeSet<>();

    for (int i = 0; i < nums.length; i++) {
      long value = nums[i];

      // ceiling(value - valueDiff) 是窗口中最小的候选值。
      Long candidate = window.ceiling(value - valueDiff);
      if (candidate != null && candidate <= value + valueDiff) {
        return true;
      }

      window.add(value);
      if (window.size() > indexDiff) {
        window.remove((long) nums[i - indexDiff]);
      }
    }
    return false;
  }
}

Explanation / 解题说明

  • TreeSet 保存最近 indexDiff 个数，并支持 O(log k) 的有序查找。
  • 对当前值 value，只需找第一个不小于 value - valueDiff 的候选值，再检查其是否不大于右边界。
  • 使用 long 避免整数加减溢出。时间复杂度为 O(n log k)，空间复杂度为 O(k)。

239. Sliding Window Maximum

Difficulty: Hard | Key Concept: sliding window, deque, monotonic queue

Original Problem

Given array nums and window size k, return an array of maximums for each sliding window.

中文翻译

给定数组 nums 和窗口大小 k，窗口从数组左侧逐步滑向右侧，返回每个窗口中的最大值。

Example

Input: nums = [1,3,-1,-3,5,3,6,7], k = 3

Output: [3,3,5,5,6,7]

Java
class Solution {
  public int[] maxSlidingWindow(int[] nums, int k) {
    int n = nums.length;
    int[] result = new int[n - k + 1];
    Deque<Integer> deque = new ArrayDeque<>();
    for (int i = 0; i < n; i++) {
      while (!deque.isEmpty() && deque.peekFirst() < i - k + 1) {
        deque.pollFirst();
      }
      while (!deque.isEmpty() && nums[deque.peekLast()] <= nums[i]) {
        deque.pollLast();
      }
      deque.offerLast(i);
      if (i >= k - 1) {
        result[i - k + 1] = nums[deque.peekFirst()];
      }
    }
    return result;
  }
}

Explanation / 解题说明

  • 使用双端队列存储索引，保持队列中元素值递减
  • 移除超出窗口的头部索引和比新元素小的尾部索引
  • 队首始终是当前窗口最大值的索引

395. Longest Substring with At Least K Repeating Characters

Difficulty: Medium | Key Concept: divide and conquer, frequency counting

Original Problem

Given string s and integer k, find the length of the longest substring in which every character appears at least k times.

中文翻译

给定字符串 s 和整数 k，找出最长连续子串，使该子串中的每一种字符都至少出现 k 次，并返回其长度。

Example

Input: s = "aaabb", k = 3

Output: 3

Explanation: "aaa" is the longest valid substring.

Java
class Solution {
  public int longestSubstring(String s, int k) {
    return divide(s.toCharArray(), 0, s.length(), k);
  }

  private int divide(char[] chars, int start, int end, int k) {
    if (end - start < k) {
      return 0;
    }

    int[] count = new int[26];
    for (int i = start; i < end; i++) {
      count[chars[i] - 'a']++;
    }

    for (int i = start; i < end; i++) {
      if (count[chars[i] - 'a'] < k) {
        // 当前字符不可能出现在合法答案中，以它为分界递归处理两侧。
        int left = divide(chars, start, i, k);
        int right = divide(chars, i + 1, end, k);
        return Math.max(left, right);
      }
    }
    return end - start;
  }
}

Explanation / 解题说明

  • 如果某个字符在当前区间中的总次数小于 k，任何包含它的子串都不合法。
  • 因此可以在该字符处切开，递归求左右区间的最优答案。
  • 最坏时间复杂度为 O(n^2)，常见情况下接近 O(n)；递归栈空间最坏为 O(n)。

480. Sliding Window Median

Difficulty: Hard | Key Concept: sliding window, heap, two heaps

Original Problem

Given array nums and window size k, return the median of each sliding window.

中文翻译

给定数组 nums 和窗口大小 k，返回窗口每次滑动后对应元素集合的中位数。

Example

Input: nums = [1,3,-1,-3,5,3,6,7], k = 3

Output: [1,-1,-1,3,5,6]

Java
class Solution {
  public double[] medianSlidingWindow(int[] nums, int k) {
    double[] result = new double[nums.length - k + 1];
    PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a,b) -> Integer.compare(b, a));
    PriorityQueue<Integer> minHeap = new PriorityQueue<>();
    for (int i = 0; i < nums.length; i++) {
      if (maxHeap.isEmpty() || nums[i] <= maxHeap.peek()) {
        maxHeap.offer(nums[i]);
      } else {
        minHeap.offer(nums[i]);
      }
      balance(maxHeap, minHeap);
      if (i >= k - 1) {
        if (k % 2 == 0) {
          result[i - k + 1] = ((double) maxHeap.peek() + minHeap.peek()) / 2;
        } else {
          result[i - k + 1] = maxHeap.peek();
        }
        int remove = nums[i - k + 1];
        if (remove <= maxHeap.peek()) {
          maxHeap.remove(remove);
        } else {
          minHeap.remove(remove);
        }
        balance(maxHeap, minHeap);
      }
    }
    return result;
  }
  
  private void balance(PriorityQueue<Integer> maxHeap, PriorityQueue<Integer> minHeap) {
    if (maxHeap.size() > minHeap.size() + 1) minHeap.offer(maxHeap.poll());
    if (minHeap.size() > maxHeap.size()) maxHeap.offer(minHeap.poll());
  }
}

Explanation / 解题说明

  • 使用大小堆维护中位数，maxHeap存储左半部分，minHeap存储右半部分
  • 移除滑动窗口滑出的元素使用remove()方法（O(n)复杂度）
  • 每次平衡两堆使maxHeap大小不小于minHeap且最多多1个

632. Smallest Range Covering Elements from K Lists

Difficulty: Hard | Key Concept: min heap, multi-list window

Original Problem

Given k sorted lists, find the smallest inclusive range that contains at least one number from each list.

中文翻译

给定 k 个已经排序的整数列表，找出一个最小闭区间，使得每个列表中都至少有一个数字落在该区间内。

Example

Input: nums = [[4,10,15,24,26],[0,9,12,20],[5,18,22,30]]

Output: [20,24]

Java
class Solution {
  public int[] smallestRange(List<List<Integer>> nums) {
    PriorityQueue<int[]> minHeap =
        new PriorityQueue<>((a, b) -> Integer.compare(a[0], b[0]));
    int currentMax = Integer.MIN_VALUE;

    for (int listIndex = 0; listIndex < nums.size(); listIndex++) {
      int value = nums.get(listIndex).get(0);
      minHeap.offer(new int[] {value, listIndex, 0});
      currentMax = Math.max(currentMax, value);
    }

    int bestStart = 0;
    int bestEnd = Integer.MAX_VALUE;

    while (minHeap.size() == nums.size()) {
      // 堆中始终保留每个列表的一个当前候选值。
      int[] current = minHeap.poll();
      int currentMin = current[0];
      int listIndex = current[1];
      int elementIndex = current[2];

      if (currentMax - currentMin < bestEnd - bestStart
          || (currentMax - currentMin == bestEnd - bestStart
              && currentMin < bestStart)) {
        bestStart = currentMin;
        bestEnd = currentMax;
      }

      int nextIndex = elementIndex + 1;
      if (nextIndex < nums.get(listIndex).size()) {
        int nextValue = nums.get(listIndex).get(nextIndex);
        minHeap.offer(new int[] {nextValue, listIndex, nextIndex});
        currentMax = Math.max(currentMax, nextValue);
      }
    }
    return new int[] {bestStart, bestEnd};
  }
}

Explanation / 解题说明

  • 堆顶是当前范围中的最小值，同时用 currentMax 记录当前最大值。
  • 弹出最小值后，只能推进它所属列表，才能尝试缩小覆盖范围。
  • 时间复杂度为 O(N log k)，空间复杂度为 O(k)，其中 N 是所有列表元素总数。

Original Problem

class Solution {
  public List<Integer> findClosestElements(int[] arr, int k, int x) {
    int left = 0;
    int right = arr.length - k;

    while (left < right) {
      int middle = left + (right - left) / 2;
      // 比较窗口两端竞争者，决定最优窗口向哪一侧收缩。
      if (x - arr[middle] > arr[middle + k] - x) {
        left = middle + 1;
      } else {
        right = middle;
      }
    }

    List<Integer> answer = new ArrayList<>();
    for (int i = left; i < left + k; i++) {
      answer.add(arr[i]);
    }
    return answer;
  }
}

683. K Empty Slots

Difficulty: Hard | Key Concept: sliding window, position-to-day transformation

Original Problem

Bulbs turn on one by one according to bulbs[day] = position. Return the earliest day when two turned-on bulbs have exactly k unlit bulbs between them, or -1 if it never happens.

中文翻译

灯泡按照 bulbs[day] = position 给出的顺序逐日点亮。返回最早满足条件的天数：存在两个已经点亮的灯泡，并且它们之间恰好有 k 个尚未点亮的灯泡。如果永远无法满足条件，返回 -1。

Input: bulbs = [1,3,2], k = 1

class Solution {
  public int kEmptySlots(int[] bulbs, int k) {
    int n = bulbs.length;
    int[] days = new int[n];

    for (int day = 0; day < n; day++) {
      // days[position] 表示该位置在第几天亮起。
      days[bulbs[day] - 1] = day + 1;
    }

    int answer = Integer.MAX_VALUE;
    int left = 0;
    int right = k + 1;

    for (int current = 1; right < n; current++) {
      if (days[current] > days[left] && days[current] > days[right]) {
        continue;
      }

      if (current == right) {
        answer = Math.min(answer, Math.max(days[left], days[right]));
      }

      // current 更早亮起，当前窗口不可能成立；从这里开始检查新窗口。
      left = current;
      right = current + k + 1;
    }
    return answer == Integer.MAX_VALUE ? -1 : answer;
  }
}

Explanation / 解题说明

  • 将按天开灯转换为 days[position] = day 后，问题变成检查固定长度位置窗口。
  • 两端亮起时，如果中间位置都更晚亮起，就得到一个合法日期。
  • 时间复杂度为 O(n)，空间复杂度为 O(n)。

Difficulty: Hard | Key Concept: sliding window, DP, prefix sum

Original Problem

Given array nums and integer k, find three non-overlapping subarrays of length k with maximum sum. Return indices of starting positions.

中文翻译

给定数组 nums 和整数 k，找出三个互不重叠且长度均为 k 的连续子数组，使它们的总和最大，并返回三个子数组的起始下标。

class Solution {
  public int[] maxSumOfThreeSubarrays(int[] nums, int k) {
    int n = nums.length;
    int[] sum = new int[n - k + 1];
    int windowSum = 0;
    for (int i = 0; i < n; i++) {
      windowSum += nums[i];
      if (i >= k - 1) {
        sum[i - k + 1] = windowSum;
        windowSum -= nums[i - k + 1];
      }
    }
    int[] left = new int[sum.length];
    int[] right = new int[sum.length];
    int best = 0;
    for (int i = 0; i < sum.length; i++) {
      if (sum[i] > sum[best]) best = i;
      left[i] = best;
    }
    best = sum.length - 1;
    for (int i = sum.length - 1; i >= 0; i--) {
      if (sum[i] >= sum[best]) best = i;
      right[i] = best;
    }
    int[] result = {-1, -1, -1};
    for (int mid = k; mid < sum.length - k; mid++) {
      int l = left[mid - k];
      int r = right[mid + k];
      if (result[0] == -1 || sum[l] + sum[mid] + sum[r] > 
          sum[result[0]] + sum[result[1]] + sum[result[2]]) {
        result = new int[]{l, mid, r};
      }
    }
    return result;
  }
}

  • 固定中间窗口位置mid，用左右最大和计算总最大值

class Solution {
  public int shortestSubarray(int[] nums, int k) {
    int n = nums.length;
    long[] prefix = new long[n + 1];
    for (int i = 0; i < n; i++) prefix[i + 1] = prefix[i] + nums[i];
    Deque<Integer> deque = new ArrayDeque<>();
    int minLen = n + 1;
    for (int i = 0; i <= n; i++) {
      while (!deque.isEmpty() && prefix[i] - prefix[deque.peekFirst()] >= k) {
        minLen = Math.min(minLen, i - deque.pollFirst());
      }
      while (!deque.isEmpty() && prefix[i] <= prefix[deque.peekLast()]) {
        deque.pollLast();
      }
      deque.offerLast(i);
    }
    return minLen == n + 1 ? -1 : minLen;
  }

  public int maxSumTwoNoOverlap(int[] nums, int firstLen, int secondLen) {
    return Math.max(helper(nums, firstLen, secondLen), helper(nums, secondLen, firstLen));
  }
  
  private int helper(int[] nums, int len1, int len2) {
    int[] sum1 = new int[nums.length - len1 + 1];
    int[] sum2 = new int[nums.length - len2 + 1];
    int windowSum = 0;
    for (int i = 0; i < nums.length; i++) {
      windowSum += nums[i];
      if (i >= len1 - 1) {
        sum1[i - len1 + 1] = windowSum;
        windowSum -= nums[i - len1 + 1];
      }
    }
    windowSum = 0;
    for (int i = 0; i < nums.length; i++) {
      windowSum += nums[i];
      if (i >= len2 - 1) {
        sum2[i - len2 + 1] = windowSum;
        windowSum -= nums[i - len2 + 1];
      }

    for (int i = 0; i + len1 <= sum2.length; i++) {
      int maxSum2 = 0;
      for (int j = i + len1; j < sum2.length; j++) {
        maxSum2 = Math.max(maxSum2, sum2[j]);
      }
      max = Math.max(max, sum1[i] + maxSum2);
    }
    return max;
  }
}

  • 计算所有长度窗口的和，然后找不重叠的最大组合

1044. Longest Duplicate Substring

Difficulty: Hard | Key Concept: sliding window, binary search, rolling hash

给定字符串 s，返回至少出现两次的最长重复子串。两次出现的位置允许重叠。

Example

Input: s = "banana"

Output: "ana"

class Solution {
  public String longestDupSubstring(String s) {
    int left = 1, right = s.length() - 1;
    String result = "";
    while (left <= right) {
      int mid = left + (right - left) / 2;
      String dup = findDuplicate(s, mid);
      if (dup != null) {
        result = dup;
        left = mid + 1;
      } else {
        right = mid - 1;
      }
    }
    return result;
  }
  
  private String findDuplicate(String s, int len) {
    Set<Long> seen = new HashSet<>();
    long hash = 0, base = 26, mod = (1L << 32);
    long power = 1;
    for (int i = 0; i < len; i++) {
      hash = (hash * base + (s.charAt(i) - 'a')) % mod;
      power = (power * base) % mod;
    }
    seen.add(hash);
    for (int i = len; i < s.length(); i++) {
      hash = (hash * base - (s.charAt(i - len) - 'a') * power % mod + mod) % mod;
      hash = (hash + (s.charAt(i) - 'a')) % mod;
      if (seen.contains(hash)) {
        return s.substring(i - len + 1, i + 1);
      }
      seen.add(hash);
    }
    return null;
  }
}

Explanation / 解题说明

  • 二分查找可能的最长重复子串长度
  • 使用滚动哈希(Rabin-Karp)在O(n)内检查是否存在重复
  • 出现哈希碰撞时需验证实际字符串（为简洁省略了碰撞验证）

Difficulty: Hard | Key Concept: sliding window, deque, DP

Original Problem

Given array nums and integer k, find max sum of non-empty subsequence where indices differ by at most k.

中文翻译

给定数组 nums 和整数 k，找出一个非空子序列，使相邻选中元素的下标差不超过 k，并返回该子序列的最大元素和。

class Solution {
  public int constrainedSubsetSum(int[] nums, int k) {
    Deque<int[]> deque = new ArrayDeque<>();
    int maxSum = nums[0];
    for (int i = 0; i < nums.length; i++) {
      int prevMax = deque.isEmpty() ? 0 : deque.peekFirst()[1];
      int curr = nums[i] + Math.max(0, prevMax);
      maxSum = Math.max(maxSum, curr);
      while (!deque.isEmpty() && deque.peekLast()[1] <= curr) {
        deque.pollLast();
      }
      deque.offerLast(new int[]{i, curr});
      while (!deque.isEmpty() && deque.peekFirst()[0] <= i - k) {
        deque.pollFirst();
      }
    }
    return maxSum;
  }
}

Explanation / 解题说明

  • 用双端队列维护长度为k窗口内DP值的最大值
  • 队列保持递减，队首始终是窗口内最大值

    Deque<Integer> deque = new ArrayDeque<>();
    int maxVal = Integer.MIN_VALUE;
    for (int j = 0; j < points.length; j++) {
      while (!deque.isEmpty() && points[j][0] - points[deque.peekFirst()][0] > k) {
        deque.pollFirst();
      }
      if (!deque.isEmpty()) {
        int i = deque.peekFirst();
        maxVal = Math.max(maxVal, points[i][1] - points[i][0] + points[j][1] + points[j][0]);
      }
      while (!deque.isEmpty() && points[deque.peekLast()][1] - points[deque.peekLast()][0] 
             <= points[j][1] - points[j][0]) {
        deque.pollLast();
      }
      deque.offerLast(j);
    }
    return maxVal;
  }
}

class Solution {
  public int minMoves(int[] nums, int k) {
    List<Integer> pos = new ArrayList<>();
    for (int i = 0; i < nums.length; i++) {
      if (nums[i] == 1) pos.add(i);
    }
    int m = pos.size();
    long[] prefix = new long[m + 1];
    for (int i = 0; i < m; i++) prefix[i + 1] = prefix[i] + pos.get(i);
    long minMoves = Long.MAX_VALUE;
    for (int i = 0; i + k <= m; i++) {
      int mid = i + k / 2;
      long leftSum = prefix[mid] - prefix[i];
      long rightSum = prefix[i + k] - prefix[mid];
      long leftCount = mid - i;
      long rightCount = i + k - mid;
      long moves = (pos.get(mid) * leftCount - leftSum) + (rightSum - pos.get(mid) * rightCount);
      if (k % 2 == 0) {
        moves -= rightCount;
      }
      minMoves = Math.min(minMoves, moves);
    }
    return (int) minMoves;
  }
}

  • k为偶数时中位数偏左，需额外调整

    if (s.charAt(n - 1) == '1') return false;

    int left = Integer.MAX_VALUE, right = 0;
    for (int num : nums) {
      left = Math.min(left, num);
      right += num;

    while (left < right) {
      int mid = left + (right - left) / 2;
      if (countSubarraysSumAtMost(nums, mid) < k) {
        left = mid + 1;

        right = mid;

        sum -= nums[left++];

    for (char c : s.toCharArray()) freq[c - 'a']++;
    for (int i = 0; i < 26; i++) if (freq[i] > 0) maxUnique++;

class Solution {
  public int[] getAverages(int[] nums, int k) {
    int n = nums.length;
    int[] answer = new int[n];
    Arrays.fill(answer, -1);

    int windowSize = 2 * k + 1;
    if (windowSize > n) {
      return answer;
    }

    long sum = 0;
    for (int i = 0; i < windowSize; i++) {
      sum += nums[i];

    }

  }

中文翻译

Input: fruits = [[2,8],[6,3],[8,6]], startPos = 5, k = 4

Output: 9

class Solution {
  public int maxTotalFruits(int[][] fruits, int startPos, int k) {
    int answer = 0;
    int sum = 0;

    for (int left = 0, right = 0; right < fruits.length; right++) {
      sum += fruits[right][1];

      while (left <= right && minSteps(fruits, left, right, startPos) > k) {
        sum -= fruits[left][1];
        left++;
      }
      answer = Math.max(answer, sum);
    }
    return answer;
  }

  private int minSteps(int[][] fruits, int left, int right, int startPos) {
    int leftPosition = fruits[left][0];
    int rightPosition = fruits[right][0];

    // 覆盖区间时，先去左端或先去右端，取较短路径。
    return rightPosition - leftPosition
        + Math.min(
            Math.abs(startPos - leftPosition),
            Math.abs(startPos - rightPosition));
  }
}

  • 选中的水果位置一定形成一个连续区间。
  • 覆盖区间 [left, right] 的最少步数是区间长度加上起点到较近端点的距离。
  • 双指针维护可在 k 步内覆盖的最大窗口。时间复杂度为 O(n)，空间复杂度为 O(1)。

2156. Find Substring With Given Hash Value

Difficulty: Hard | Key Concept: rolling hash, reverse sliding window

Original Problem

Given string s, integers power, modulo, k, and hashValue, return the first substring of length k whose polynomial hash equals hashValue.

中文翻译

给定字符串 s 和整数 power、modulo、k、hashValue，按照题目定义的多项式哈希公式，返回第一个哈希值等于 hashValue 的长度为 k 的子串。

Example

Input: s = "leetcode", power = 7, modulo = 20, k = 2, hashValue = 0

Output: "ee"

Java
class Solution {
  public String subStrHash(
      String s, int power, int modulo, int k, int hashValue) {
    int n = s.length();
    long hash = 0;
    long highestPower = 1;

    for (int i = n - 1; i >= n - k; i--) {
      hash = (hash * power + valueOf(s.charAt(i))) % modulo;
      if (i > n - k) {
        highestPower = highestPower * power % modulo;
      }
    }

    int answerStart = hash == hashValue ? n - k : -1;

    for (int i = n - k - 1; i >= 0; i--) {
      // 先移除旧窗口最右字符，再把新字符加入最低次项位置。
      hash =
          (hash - valueOf(s.charAt(i + k)) * highestPower % modulo + modulo)
              % modulo;
      hash = (hash * power + valueOf(s.charAt(i))) % modulo;

      if (hash == hashValue) {
        answerStart = i;
      }
    }
    return s.substring(answerStart, answerStart + k);
  }

  private int valueOf(char character) {
    return character - 'a' + 1;
  }
}

Explanation / 解题说明

  • 题目哈希公式适合从右向左滑动，避免计算模逆元。
  • 每次删除离开窗口的最高次项，再将哈希整体乘以 power 并加入新字符。
  • 时间复杂度为 O(n)，空间复杂度为 O(1)。

    }

  }
}

2524. Maximum Frequency Score of a Subarray

Difficulty: Hard | Key Concept: sliding window, HashMap, fast power

Original Problem

Given array nums and integer k, define the frequency score of an array as the sum of each distinct value raised to its frequency, modulo 1_000_000_007. Return the maximum score among subarrays of length k.

中文翻译

给定数组 nums 和整数 k。数组的频率分数定义为：对每个不同数值计算“数值的出现次数次方”，将结果求和后对 1_000_000_007 取模。返回所有长度为 k 的连续子数组中的最大频率分数。

Example

Input: nums = [1,1,1,2,1,2], k = 3

Output: 5

Explanation: Subarray [2,1,2] has score 2^2 + 1^1 = 5.

Java
class Solution {
  private static final int MOD = 1_000_000_007;

  public int maxFrequencyScore(int[] nums, int k) {
    Map<Integer, Integer> frequency = new HashMap<>();
    for (int i = 0; i < k; i++) {
      frequency.put(nums[i], frequency.getOrDefault(nums[i], 0) + 1);
    }

    long currentScore = 0;
    for (Map.Entry<Integer, Integer> entry : frequency.entrySet()) {
      currentScore =
          (currentScore + fastPower(entry.getKey(), entry.getValue())) % MOD;
    }

    long answer = currentScore;
    for (int right = k; right < nums.length; right++) {
      int leaving = nums[right - k];
      int entering = nums[right];

      if (leaving != entering) {
        // x^(count + 1) - x^count = (x - 1) * x^count。
        int enteringCount = frequency.getOrDefault(entering, 0);
        long enteringDelta =
            enteringCount == 0
                ? entering
                : (long) (entering - 1) * fastPower(entering, enteringCount)
                    % MOD;
        currentScore = (currentScore + enteringDelta) % MOD;

        int leavingCount = frequency.get(leaving);
        long leavingDelta =
            leavingCount == 1
                ? leaving
                : (long) (leaving - 1) * fastPower(leaving, leavingCount - 1)
                    % MOD;
        currentScore = (currentScore - leavingDelta + MOD) % MOD;

        frequency.put(entering, enteringCount + 1);
        if (leavingCount == 1) {
          frequency.remove(leaving);
        } else {
          frequency.put(leaving, leavingCount - 1);
        }
      }
      answer = Math.max(answer, currentScore);
    }
    return (int) answer;
  }

  private long fastPower(long base, int exponent) {
    long answer = 1;
    while (exponent > 0) {
      if ((exponent & 1) == 1) {
        answer = answer * base % MOD;
      }
      base = base * base % MOD;
      exponent >>= 1;
    }
    return answer;
  }
}

Explanation / 解题说明

  • 分数不是频率平方和，而是各个不同值的 value^frequency 之和。
  • 窗口滑动时只更新离开值和进入值的贡献，快速幂用于计算变化量。
  • 时间复杂度为 O(n log k)，空间复杂度为 O(k)。

2528. Maximize the Minimum Powered City

Difficulty: Hard | Key Concept: binary search, greedy, difference array

Original Problem

Each city contains some power stations. A station powers cities within radius r. Add at most k stations to maximize the minimum power among all cities.

中文翻译

每座城市包含若干电站，每个电站可以为距离不超过 r 的城市供电。最多可以新增 k 个电站，返回合理安排新增电站后，所有城市供电量最小值能够达到的最大值。

Input: stations = [1,2,4,5,0], r = 1, k = 2

Output: 5

class Solution {
  private long[] basePower;
  private long[] difference;
  private int cityCount;

  public long maxPower(int[] stations, int r, int k) {
    cityCount = stations.length;
    difference = new long[cityCount + 1];
    basePower = new long[cityCount];

    for (int city = 0; city < cityCount; city++) {
      int left = Math.max(0, city - r);
      int right = Math.min(cityCount - 1, city + r);
      difference[left] += stations[city];
      difference[right + 1] -= stations[city];
    }

    long running = 0;
    for (int city = 0; city < cityCount; city++) {
      running += difference[city];
      basePower[city] = running;
    }

    long left = 0;
    long right = (long) 1e15;
    while (left < right) {
      long middle = left + (right - left + 1) / 2;
      if (canReach(middle, r, k)) {
        left = middle;
      } else {
        right = middle - 1;
      }
    }
    return left;
  }

  private boolean canReach(long target, int r, int k) {
    Arrays.fill(difference, 0);
    long activeAddedPower = 0;
    long usedStations = 0;

    for (int city = 0; city < cityCount; city++) {
      activeAddedPower += difference[city];
      long need = target - (basePower[city] + activeAddedPower);
      if (need <= 0) {
        continue;
      }

      usedStations += need;
      if (usedStations > k) {
        return false;
      }

      // 为尽量覆盖右侧城市，把新增电站放到 city + r 的最远合法位置。
      activeAddedPower += need;
      int expiration = Math.min(cityCount, city + 2 * r + 1);
      difference[expiration] -= need;
    }
    return true;
  }
}

Explanation / 解题说明

  • 先用差分数组计算每个城市的初始供电量。
  • 二分目标最小供电量。检查可行性时，从左到右贪心补足当前城市，并把新增电站放在最靠右的位置。
  • 每次检查为 O(n)，总时间复杂度为 O(n log M)，空间复杂度为 O(n)。

        left = -1;
        continue;

      }
    }
    return answer;
  }

2902. Count of Sub-Multisets With Bounded Sum

Difficulty: Hard | Key Concept: bounded knapsack, prefix optimization

Original Problem

Given a multiset represented by list nums and integers l and r, count sub-multisets whose sum lies in [l, r]. Return the answer modulo 1_000_000_007.

中文翻译

给定由列表 nums 表示的多重集合以及整数 l 和 r，返回元素和位于闭区间 [l, r] 内的子多重集合数量。结果对 1_000_000_007 取模。

Example

Input: nums = [1,2,2,3], l = 6, r = 6

Output: 1

Java
class Solution {
  private static final int MOD = 1_000_000_007;

  public int countSubMultisets(List<Integer> nums, int l, int r) {
    Map<Integer, Integer> frequency = new HashMap<>();
    int total = 0;

    for (int value : nums) {
      total += value;
      if (value <= r) {
        frequency.put(value, frequency.getOrDefault(value, 0) + 1);
      }
    }

    if (total < l) {
      return 0;
    }
    r = Math.min(r, total);

    int[] dp = new int[r + 1];
    // 选择 0 个、1 个、...、zeroCount 个零都不会改变总和。
    dp[0] = frequency.getOrDefault(0, 0) + 1;
    frequency.remove(0);

    int reachableSum = 0;
    for (Map.Entry<Integer, Integer> entry : frequency.entrySet()) {
      int value = entry.getKey();
      int count = entry.getValue();
      reachableSum = Math.min(r, reachableSum + value * count);

      for (int sum = value; sum <= reachableSum; sum++) {
        dp[sum] = (dp[sum] + dp[sum - value]) % MOD;
      }

      // 删除使用超过 count 个当前值的方案。
      int overflow = value * (count + 1);
      for (int sum = reachableSum; sum >= overflow; sum--) {
        dp[sum] = (dp[sum] - dp[sum - overflow] + MOD) % MOD;
      }
    }

    int answer = 0;
    for (int sum = l; sum <= r; sum++) {
      answer = (answer + dp[sum]) % MOD;
    }
    return answer;
  }
}

Explanation / 解题说明

  • 先压缩相同数字的频率，将问题转为多重背包。
  • dp[sum] 表示组成当前和的子多重集合数量，循环更新后再扣除使用次数超限的方案。
  • 时间复杂度为 O(n + U * r)，空间复杂度为 O(r)，其中 U 是不同正整数数量。

  public int maxFrequencyScore(int[] nums, long k) {
    Arrays.sort(nums);
    int n = nums.length;
    long[] prefix = new long[n + 1];
    for (int i = 0; i < n; i++) prefix[i + 1] = prefix[i] + nums[i];
    int left = 0, maxFreq = 0;
    for (int right = 0; right < n; right++) {
      int mid = (left + right) / 2;

    }

3013. Divide an Array Into Subarrays With Minimum Cost II

Difficulty: Hard | Key Concept: sliding window, heap, multiset

Original Problem

Given array nums and integers k, dist, partition array into k subarrays with cost = sum of first elements. Min cost.

中文翻译

给定数组 nums 以及整数 k 和 dist，需要将数组划分为 k 个非空连续子数组。总成本为每个子数组首元素之和，并且后续子数组起点之间需要满足距离限制。返回最小成本。

Example

Input: nums = [1,3,2,6,4,2], k = 3, dist = 3

Output: 5

Java
class Solution {
  public long minimumCost(int[] nums, int k, int dist) {
    SelectionWindow window = new SelectionWindow(k - 1);

    // 第二个到第 k 个子数组的起点必须落在跨度不超过 dist 的窗口中。
    for (int index = 1; index <= dist + 1; index++) {
      window.add(nums[index]);
    }

    long answer = nums[0] + window.selectedSum();
    for (int right = dist + 2; right < nums.length; right++) {
      window.remove(nums[right - dist - 1]);
      window.add(nums[right]);
      answer = Math.min(answer, nums[0] + window.selectedSum());
    }

    return answer;
  }

  private static class SelectionWindow {
    private final int targetSize;
    private final TreeMap<Integer, Integer> selected = new TreeMap<>();
    private final TreeMap<Integer, Integer> remaining = new TreeMap<>();
    private int selectedSize;
    private long selectedSum;

    SelectionWindow(int targetSize) {
      this.targetSize = targetSize;
    }

    long selectedSum() {
      return selectedSum;
    }

    void add(int value) {
      addCount(remaining, value);
      rebalance();
    }

    void remove(int value) {
      if (selected.containsKey(value)) {
        removeCount(selected, value);
        selectedSize--;
        selectedSum -= value;
      } else {
        removeCount(remaining, value);
      }
      rebalance();
    }

    private void rebalance() {
      while (selectedSize < targetSize && !remaining.isEmpty()) {
        move(remaining, selected, remaining.firstKey(), true);
      }
      while (selectedSize > targetSize) {
        move(selected, remaining, selected.lastKey(), false);
      }

      // selected始终保存窗口内最小的 targetSize 个值。
      while (!selected.isEmpty()
          && !remaining.isEmpty()
          && selected.lastKey() > remaining.firstKey()) {
        int larger = selected.lastKey();
        int smaller = remaining.firstKey();
        move(selected, remaining, larger, false);
        move(remaining, selected, smaller, true);
      }
    }

    private void move(
        TreeMap<Integer, Integer> from,
        TreeMap<Integer, Integer> to,
        int value,
        boolean intoSelected) {
      removeCount(from, value);
      addCount(to, value);
      if (intoSelected) {
        selectedSize++;
        selectedSum += value;
      } else {
        selectedSize--;
        selectedSum -= value;
      }
    }

    private void addCount(TreeMap<Integer, Integer> map, int value) {
      map.put(value, map.getOrDefault(value, 0) + 1);
    }

    private void removeCount(TreeMap<Integer, Integer> map, int value) {
      int count = map.get(value);
      if (count == 1) {
        map.remove(value);
      } else {
        map.put(value, count - 1);
      }
    }
  }
}

Explanation / 解题说明

  • 第一个子数组的成本固定为nums[0]，其余成本来自长度为dist + 1的候选窗口中最小的k - 1个值。
  • 两个TreeMap分别维护已选值与剩余值，并支持重复元素。
  • 每次窗口滑动后重新平衡，使selected始终保存最小的k - 1个候选值。时间复杂度为O(n log dist)。

    int n = s.length();
    int[] leftBound = new int[n];
    int[] cnt = new int[2];
    int left = 0;
    for (int right = 0; right < n; right++) {
      cnt[s.charAt(right) - '0']++;
      while (cnt[0] > k && cnt[1] > k) {

      leftBound[right] = left;

          lo = mid + 1;
        }

  public int minConnectedGroups(int[][] intervals, int[] newInterval) {
    Arrays.sort(intervals, (a,b) -> a[0] - b[0]);
    List<int[]> merged = new ArrayList<>();
    for (int[] inv : intervals) {
      if (merged.isEmpty() || merged.get(merged.size() - 1)[1] < inv[0]) {
        merged.add(new int[]{inv[0], inv[1]});

        merged.get(merged.size() - 1)[1] = Math.max(merged.get(merged.size() - 1)[1], inv[1]);

      while (merged.get(right)[0] > merged.get(left)[1] + 1 + (end - start)) {
        left++;
      }
      maxConnect = Math.max(maxConnect, right - left + 1);
    }
    return groups - maxConnect + 1;

  public int maxFreeTime(

    int meetingCount = startTime.length;

class Solution {
  public int maxDifference(String s, int k) {
    int n = s.length();
    int answer = Integer.MIN_VALUE;

    for (int oddDigit = 0; oddDigit < 5; oddDigit++) {
      for (int evenDigit = 0; evenDigit < 5; evenDigit++) {
        if (oddDigit == evenDigit) {
          continue;
        }

        int[] oddPrefix = new int[n + 1];
        int[] evenPrefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
          int digit = s.charAt(i) - '0';
          oddPrefix[i + 1] = oddPrefix[i] + (digit == oddDigit ? 1 : 0);
          evenPrefix[i + 1] = evenPrefix[i] + (digit == evenDigit ? 1 : 0);
        }

        int inf = 1 << 29;
        int[][] minPrefix = {{inf, inf}, {inf, inf}};
        int left = 0;

        for (int end = 1; end <= n; end++) {
          // 只加入长度足够且能保证 evenDigit 在子串中至少出现一次的前缀。
          while (left <= end - k && evenPrefix[left] < evenPrefix[end]) {
            int oddParity = oddPrefix[left] & 1;
            int evenParity = evenPrefix[left] & 1;
            minPrefix[oddParity][evenParity] =
                Math.min(
                    minPrefix[oddParity][evenParity],
                    oddPrefix[left] - evenPrefix[left]);
            left++;
          }

          int neededOddParity = (oddPrefix[end] & 1) ^ 1;
          int neededEvenParity = evenPrefix[end] & 1;
          int best = minPrefix[neededOddParity][neededEvenParity];
          if (best != inf) {
            answer =
                Math.max(answer, oddPrefix[end] - evenPrefix[end] - best);
          }
        }
      }
    }
    return answer;
  }
}

  • 枚举有奇数频率的字符和有正偶数频率的字符。
  • 使用前缀计数奇偶性作为状态，并维护合法左前缀中的最小差值。
  • 时间复杂度为 O(25 * n)，空间复杂度为 O(n)。

3505. Minimum Operations to Make Elements Within K Subarrays Equal

Difficulty: Hard | Key Concept: sliding window, DP, binary search

Original Problem

Given array nums and integers x and k, return the minimum operations needed to obtain at least k non-overlapping subarrays of size exactly x, where all elements within each selected subarray are equal. One operation increases or decreases one element by 1.

中文翻译

class Solution {
  public long minOperations(int[] nums, int x, int k) {
    int n = nums.length;
    long[] windowCost = new long[n - x + 1];
    MedianWindow window = new MedianWindow();

    for (int right = 0; right < n; right++) {
      window.add(nums[right]);
      if (right >= x) {
        window.remove(nums[right - x]);
      }
      if (right >= x - 1) {
        windowCost[right - x + 1] = window.cost();
      }
    }

    long inf = Long.MAX_VALUE / 4;
    long[] previous = new long[n + 1];

    for (int selected = 1; selected <= k; selected++) {
      long[] current = new long[n + 1];
      Arrays.fill(current, inf);

      for (int prefixLength = 1; prefixLength <= n; prefixLength++) {
        current[prefixLength] = current[prefixLength - 1];
        if (prefixLength >= x && previous[prefixLength - x] != inf) {
          current[prefixLength] =
              Math.min(
                  current[prefixLength],
                  previous[prefixLength - x] + windowCost[prefixLength - x]);
        }
      }
      previous = current;
    }
    return previous[n];
  }

  private static class MedianWindow {
    private final TreeMap<Integer, Integer> lower = new TreeMap<>();
    private final TreeMap<Integer, Integer> upper = new TreeMap<>();
    private int lowerSize;
    private int upperSize;
    private long lowerSum;
    private long upperSum;

    void add(int value) {
      if (lower.isEmpty() || value <= lower.lastKey()) {
        addLower(value);
      } else {
        addUpper(value);
      }
      rebalance();
    }

    void remove(int value) {
      if (lower.containsKey(value)) {
        removeLower(value);
      } else {
        removeUpper(value);
      }
      rebalance();
    }

    long cost() {
      long median = lower.lastKey();
      return median * lowerSize - lowerSum + upperSum - median * upperSize;
    }

    private void rebalance() {
      while (lowerSize > upperSize + 1) {
        int value = lower.lastKey();
        removeLower(value);
        addUpper(value);
      }
      while (lowerSize < upperSize) {
        int value = upper.firstKey();
        removeUpper(value);
        addLower(value);
      }
    }

    private void addLower(int value) {
      lower.put(value, lower.getOrDefault(value, 0) + 1);
      lowerSize++;
      lowerSum += value;
    }

    private void addUpper(int value) {
      upper.put(value, upper.getOrDefault(value, 0) + 1);
      upperSize++;
      upperSum += value;
    }

    private void removeLower(int value) {
      removeOne(lower, value);
      lowerSize--;
      lowerSum -= value;
    }

    private void removeUpper(int value) {
      removeOne(upper, value);
      upperSize--;
      upperSum -= value;
    }

    private void removeOne(TreeMap<Integer, Integer> values, int value) {
      int count = values.get(value);
      if (count == 1) {
        values.remove(value);
      } else {
        values.put(value, count - 1);
      }
    }
  }
}

Explanation / 解题说明

  • 长度为 x 的窗口变为全相等时，变为中位数的成本最小。
  • 双 TreeMap 在线维护每个窗口的中位数和成本，再用 DP 选择 k 个互不重叠窗口。
  • 时间复杂度为 O(n log x + n * k)，空间复杂度为 O(n + x)。

  public int countPartitions(int[] nums, int k) {
    int mod = 1_000_000_007;
    int n = nums.length;
    long[] dp = new long[n + 1];
    long[] prefix = new long[n + 1];
    Deque<Integer> minDeque = new ArrayDeque<>();
    Deque<Integer> maxDeque = new ArrayDeque<>();
    int left = 0;

    dp[0] = 1;
    prefix[0] = 1;

    for (int right = 0; right < n; right++) {
      while (!minDeque.isEmpty() && nums[minDeque.peekLast()] >= nums[right]) {
        minDeque.pollLast();
      }

        maxDeque.pollLast();

        left++;

    }

    List<Integer> positions = new ArrayList<>();
    List<Integer> values = new ArrayList<>();

    for (int i = 0; i < n; i++) {
      if (isPrime[nums[i]]) {
        positions.add(i);
        values.add(nums[i]);
      }
    }

    long answer = 0;
    int left = 0;
    Deque<Integer> minDeque = new ArrayDeque<>();
    Deque<Integer> maxDeque = new ArrayDeque<>();

    for (int right = 0; right < values.size(); right++) {
      while (!minDeque.isEmpty() && values.get(minDeque.peekLast()) >= values.get(right)) {
        minDeque.pollLast();
      }
      minDeque.offerLast(right);

      while (!maxDeque.isEmpty() && values.get(maxDeque.peekLast()) <= values.get(right)) {
        maxDeque.pollLast();
      }
      maxDeque.offerLast(right);

      while (values.get(maxDeque.peekFirst()) - values.get(minDeque.peekFirst()) > k) {
        if (minDeque.peekFirst() == left) {
          minDeque.pollFirst();
        }
        if (maxDeque.peekFirst() == left) {
          maxDeque.pollFirst();
        }
        left++;
      }

      if (right - left >= 1) {
        int previousPrime = left == 0 ? -1 : positions.get(left - 1);
        long startChoices = positions.get(right - 1) - previousPrime;
        int nextPrime =
            right + 1 < positions.size() ? positions.get(right + 1) : nums.length;
        long endChoices = nextPrime - positions.get(right);
        answer += startChoices * endChoices;
      }
    }
    return answer;
  }

  private boolean[] buildPrimeTable(int limit) {
    boolean[] isPrime = new boolean[limit + 1];
    Arrays.fill(isPrime, true);
    if (limit >= 0) {
      isPrime[0] = false;
    }
    if (limit >= 1) {
      isPrime[1] = false;
    }

    for (int value = 2; value * value <= limit; value++) {
      if (!isPrime[value]) {
        continue;
      }
      for (int multiple = value * value; multiple <= limit; multiple += value) {
        isPrime[multiple] = false;
      }
    }
    return isPrime;
  }
}

Explanation / 解题说明

  • 先筛出质数，再将原数组压缩为质数位置和值。
  • 双端队列维护当前质数窗口的最大值和最小值，确保质数差不超过 k。
  • 按可选左右边界数量累计答案。时间复杂度为 O(n log log V)。

    long[] originalProfit = new long[n + 1];
    long[] pricePrefix = new long[n + 1];

    for (int i = 0; i < n; i++) {
      originalProfit[i + 1] =
          originalProfit[i] + (long) strategy[i] * prices[i];
      pricePrefix[i + 1] = pricePrefix[i] + prices[i];

3672. Sum of Weighted Modes in Subarrays

Difficulty: Medium | Key Concept: sliding window, hashmap

Original Problem

Given array nums and integer k, compute the mode of each subarray of length k. The mode is the most frequent value, breaking ties by choosing the smallest value. Its weight is mode * frequency(mode). Return the sum of weights across all windows.

中文翻译

给定数组 nums 和整数 k。对于每个长度为 k 的连续子数组，找出出现次数最多的众数；如果有多个众数，选择数值最小的一个。窗口权重为“众数乘以其频率”，返回所有窗口权重之和。

Example

Input: nums = [1,2,2,3], k = 3

Output: 8

Java
class Solution {
  public long modeWeight(int[] nums, int k) {
    Map<Integer, Integer> frequency = new HashMap<>();
    PriorityQueue<int[]> candidates =
        new PriorityQueue<>(
            (a, b) ->
                a[0] != b[0]
                    ? Integer.compare(b[0], a[0])
                    : Integer.compare(a[1], b[1]));

    for (int i = 0; i < k; i++) {
      add(nums[i], frequency, candidates);
    }

    long answer = currentWeight(frequency, candidates);
    for (int right = k; right < nums.length; right++) {
      remove(nums[right - k], frequency, candidates);
      add(nums[right], frequency, candidates);
      answer += currentWeight(frequency, candidates);
    }
    return answer;
  }

  private void add(
      int value,
      Map<Integer, Integer> frequency,
      PriorityQueue<int[]> candidates) {
    int newCount = frequency.getOrDefault(value, 0) + 1;
    frequency.put(value, newCount);
    candidates.offer(new int[] {newCount, value});
  }

  private void remove(
      int value,
      Map<Integer, Integer> frequency,
      PriorityQueue<int[]> candidates) {
    int newCount = frequency.get(value) - 1;
    if (newCount == 0) {
      frequency.remove(value);
    } else {
      frequency.put(value, newCount);
      candidates.offer(new int[] {newCount, value});
    }
  }

  private long currentWeight(
      Map<Integer, Integer> frequency,
      PriorityQueue<int[]> candidates) {
    while (true) {
      int[] candidate = candidates.peek();
      if (frequency.getOrDefault(candidate[1], 0) == candidate[0]) {
        return (long) candidate[0] * candidate[1];
      }
      candidates.poll();
    }
  }
}

Explanation / 解题说明

  • 哈希表记录窗口频率，优先队列按“频率降序、数值升序”保存众数候选。
  • 使用延迟删除跳过已经过期的候选状态。
  • 时间复杂度为 O(n log n)，空间复杂度为 O(n)。

class Solution {
  public int distinctPoints(String s, int k) {
    int n = s.length();
    int totalX = 0;
    int totalY = 0;

    int removedX = 0;
    int removedY = 0;
    Set<Long> finalPoints = new HashSet<>();

    for (int i = 0; i < n; i++) {
      int[] direction = directionOf(s.charAt(i));
      totalX += direction[0];
      totalY += direction[1];
    }

    for (int right = 0; right < n; right++) {
      int[] added = directionOf(s.charAt(right));
      removedX += added[0];
      removedY += added[1];

      if (right >= k) {
        int[] expired = directionOf(s.charAt(right - k));
        removedX -= expired[0];
        removedY -= expired[1];
      }

      if (right >= k - 1) {
        int finalX = totalX - removedX;
        int finalY = totalY - removedY;
        finalPoints.add(encode(finalX, finalY));
      }
    }

    if (move == 'L') {
      return new int[] {-1, 0};
    }
    return new int[] {1, 0};
  }

    return ((long) x << 32) ^ (y & 0xffffffffL);
  }

3768. Minimum Inversion Count in Subarrays of Fixed Length

Difficulty: Hard | Key Concept: sliding window, Fenwick tree

Original Problem

Given array nums and integer k, find min inversion count among all subarrays length k.

中文翻译

给定数组 nums 和整数 k，逆序对定义为满足 i < j 且 nums[i] > nums[j] 的下标对。返回所有长度为 k 的连续子数组中的最小逆序对数量。

Example

Input: nums = [3,1,2,5,4], k = 3

Output: 0

Java
class Solution {
  public long minInversionCount(int[] nums, int k) {
    int[] sorted = nums.clone();
    Arrays.sort(sorted);

    FenwickTree bit = new FenwickTree(sorted.length);
    long inversions = 0;

    // 先建立第一个长度为 k 的窗口。
    // 新增 nums[i] 时，只统计已有元素中比它大的元素数量。
    for (int i = 0; i < k; i++) {
      int rank = rankOf(sorted, nums[i]);
      inversions += i - bit.query(rank);
      bit.add(rank, 1);
    }

    long answer = inversions;
    for (int right = k; right < nums.length; right++) {
      int removedRank = rankOf(sorted, nums[right - k]);

      // 左端元素离开窗口后，删除所有以它作为左侧较大值的逆序对。
      bit.add(removedRank, -1);
      inversions -= bit.query(removedRank - 1);

      int addedRank = rankOf(sorted, nums[right]);

      // 新元素位于窗口最右侧，新增逆序对来自窗口中比它大的元素。
      inversions += (k - 1) - bit.query(addedRank);
      bit.add(addedRank, 1);

      answer = Math.min(answer, inversions);
    }
    return answer;
  }

  private int rankOf(int[] sorted, int value) {
    return Arrays.binarySearch(sorted, value) + 1;
  }

  private static class FenwickTree {
    private final int[] tree;

    FenwickTree(int size) {
      tree = new int[size + 1];
    }

    void add(int index, int delta) {
      for (int i = index; i < tree.length; i += i & -i) {
        tree[i] += delta;
      }
    }

    int query(int index) {
      int sum = 0;
      for (int i = index; i > 0; i -= i & -i) {
        sum += tree[i];
      }
      return sum;
    }
  }
}

Explanation / 解题说明

  • 用排序后的下标做离散化，Fenwick Tree维护窗口内每个值出现次数。
  • 加入右端元素时统计已有元素中比它大的数量；移除左端元素时扣掉比它小的数量。
  • 每次窗口滑动只做两次树状数组查询和更新，时间复杂度为 O(n log n)。

class Solution {
  public int minLength(int[] nums, int k) {
    Map<Integer, Integer> frequency = new HashMap<>();
    long distinctSum = 0;
    int left = 0;
    int answer = Integer.MAX_VALUE;

    for (int right = 0; right < nums.length; right++) {
      int value = nums[right];
      int count = frequency.getOrDefault(value, 0) + 1;

3845. Maximum Subarray XOR with Bounded Range

Difficulty: Hard | Key Concept: sliding window, bit trie

Original Problem

Given non-negative array nums and integer k, select a subarray whose maximum and minimum values differ by at most k. Return the maximum possible bitwise XOR of the selected subarray.

中文翻译

给定非负整数数组 nums 和整数 k，选择一个连续子数组，使其中最大值与最小值之差不超过 k。返回所选子数组全部元素按位异或后能够得到的最大值。

Example

Input: nums = [5,4,5,6], k = 2

Output: 7

class Solution {
  public int maxXor(int[] nums, int k) {
    int n = nums.length;
    int[] prefixXor = new int[n + 1];
    Trie trie = new Trie();
    Deque<Integer> minDeque = new ArrayDeque<>();
    Deque<Integer> maxDeque = new ArrayDeque<>();

    int left = 0;
    int answer = 0;
    trie.add(0, 1);

    for (int right = 0; right < n; right++) {
      prefixXor[right + 1] = prefixXor[right] ^ nums[right];

      while (!minDeque.isEmpty() && nums[minDeque.peekLast()] >= nums[right]) {
        minDeque.pollLast();
      }
      minDeque.offerLast(right);

      while (!maxDeque.isEmpty() && nums[maxDeque.peekLast()] <= nums[right]) {
        maxDeque.pollLast();
      }
      maxDeque.offerLast(right);

      // 保持窗口 [left, right] 满足 max - min <= k。
      // Trie中保存的是所有仍可作为子数组起点的 prefixXor。
      while (nums[maxDeque.peekFirst()] - nums[minDeque.peekFirst()] > k) {
        trie.add(prefixXor[left], -1);
        if (minDeque.peekFirst() == left) {
          minDeque.pollFirst();
        }
        if (maxDeque.peekFirst() == left) {
          maxDeque.pollFirst();
        }
        left++;
      }

      answer = Math.max(answer, trie.maxXor(prefixXor[right + 1]));
      trie.add(prefixXor[right + 1], 1);
    }

    return answer;
  }

  private static class Trie {
    private final Node root = new Node();

    void add(int value, int delta) {
      Node node = root;
      node.count += delta;

      for (int bit = 30; bit >= 0; bit--) {
        int currentBit = (value >>> bit) & 1;
        if (node.children[currentBit] == null) {
          node.children[currentBit] = new Node();
        }
        node = node.children[currentBit];
        node.count += delta;
      }
    }

    int maxXor(int value) {
      Node node = root;
      int result = 0;

      for (int bit = 30; bit >= 0; bit--) {
        int currentBit = (value >>> bit) & 1;
        int preferredBit = currentBit ^ 1;

        if (node.children[preferredBit] != null
            && node.children[preferredBit].count > 0) {
          result |= 1 << bit;
          node = node.children[preferredBit];
        } else {
          node = node.children[currentBit];
        }
      }
      return result;
    }

    private static class Node {
      Node[] children = new Node[2];
      int count;
    }
  }
}

Explanation / 解题说明

  • 两个单调队列维护当前窗口的最大值和最小值，左边界移动后窗口始终满足max - min <= k。
  • 前缀异或满足xor(left..right) = prefixXor[left] XOR prefixXor[right + 1]。
  • Trie保存合法起点的前缀异或值，并按位贪心找到最大异或结果；总复杂度为O(n log V)。

Given request records [user, time] and integers k and window, drop as few requests as needed so that no user makes more than k requests in any inclusive interval [t, t + window]. Return the maximum number of requests that can remain.

中文翻译

给定请求记录 [user, time] 以及整数 k 和 window。可以丢弃任意请求，需要保证不存在某位用户在任意闭区间 [t, t + window] 内发出超过 k 个请求。返回最多可以保留的请求数量。

Example

Input: requests = [[1,1],[2,1],[1,7],[2,8]], k = 1, window = 4

Output: 4

Java
class Solution {
  public int maxRequests(int[][] requests, int k, int window) {
    Arrays.sort(requests, (a, b) -> Integer.compare(a[1], b[1]));
    Map<Integer, Deque<Integer>> keptTimes = new HashMap<>();
    int accepted = 0;

    for (int[] request : requests) {
      int user = request[0];
      int time = request[1];

      if (!keptTimes.containsKey(user)) {
        keptTimes.put(user, new ArrayDeque<>());
      }

      Deque<Integer> times = keptTimes.get(user);

      // 区间是闭区间：time - window 时刻的请求仍属于当前限制窗口。
      while (!times.isEmpty() && times.peekFirst() < time - window) {
        times.pollFirst();
      }

      if (times.size() < k) {
        times.offerLast(time);
        accepted++;
      }
    }

    return accepted;
  }
}

Explanation / 解题说明

  • 按时间顺序处理请求，每个用户单独维护已接受请求的时间队列。
  • 新请求到达时，先移除已经离开闭区间窗口的旧请求。
  • 若该用户窗口内仍少于k个已接受请求，就保留当前请求；否则丢弃。

3859. Count Subarrays With K Distinct Integers

Difficulty: Medium | Key Concept: sliding window, two pointers

Original Problem

Given array nums and integers k and m, count subarrays that contain exactly k distinct integers and in which every distinct integer appears at least m times.

中文翻译

给定数组 nums 以及整数 k 和 m，返回连续子数组数量，使子数组恰好包含 k 种不同整数，并且每一种出现过的整数都至少出现 m 次。

Example

Input: nums = [1,2,1,2,2], k = 2, m = 2

Output: 2

Java
class Solution {
  public long countSubarrays(int[] nums, int k, int m) {
    return countAtLeast(nums, k, k, m) - countAtLeast(nums, k + 1, k, m);
  }

  private long countAtLeast(int[] nums, int distinctNeeded, int frequentNeeded, int m) {
    Map<Integer, Integer> frequency = new HashMap<>();
    int left = 0;
    int frequentValues = 0;
    long answer = 0;

    for (int value : nums) {
      int count = frequency.getOrDefault(value, 0) + 1;
      frequency.put(value, count);
      if (count == m) {
        frequentValues++;
      }

      // 收缩到刚好不满足条件；此时所有更靠左的起点都满足条件。
      while (frequency.size() >= distinctNeeded && frequentValues >= frequentNeeded) {
        int leftValue = nums[left++];
        int leftCount = frequency.get(leftValue);

        if (leftCount == m) {
          frequentValues--;
        }
        if (leftCount == 1) {
          frequency.remove(leftValue);
        } else {
          frequency.put(leftValue, leftCount - 1);
        }
      }

      answer += left;
    }

    return answer;
  }
}

Explanation / 解题说明

  • countAtLeast(d, f)统计至少有d种不同值、且至少有f种值出现不少于m次的子数组数量。
  • “恰好k种且每种至少m次” = countAtLeast(k, k) - countAtLeast(k + 1, k)。
  • 每个右边界只会推动左边界向前移动，总复杂度为O(n)。
