1. LRU Cache #146

Problem in brief: Implement a fixed-capacity cache where every successful get or put marks a key as recently used, and eviction removes the least recently used key.

Representative example: capacity=2; put(1,1), put(2,2), get(1)->1; put(3,3) evicts key 2; get(2)->-1.

1

中文复述: 设计 cache, 支持 get/put, 容量满时淘汰最近最少使用 key.

2

Brute Force: 用 list 存顺序, 查找/移动/删除 O(n).

3

Optimal: HashMap 找 node, Doubly Linked List 维护 recency. head.next 最新, tail.prev 最旧.

4

Java Core:

Java
class LRUCache {
    class Node {
        int key, val;
        Node prev, next;
        Node(int k, int v) {
            key = k;
            val = v;
        }
    }
    private int cap;
    private Map<Integer, Node> map = new HashMap<>();
    private Node head = new Node(0, 0), tail = new Node(0, 0);
    public LRUCache(int capacity) {
        cap = capacity;
        head.next = tail;
        tail.prev = head;
    }
    public int get(int key) {
        if (!map.containsKey(key)) {
            return -1;
        }
        Node n = map.get(key);
        remove(n);
        addFirst(n);
        return n.val;
    }
    public void put(int key, int value) {
        if (map.containsKey(key)) {
            Node n = map.get(key);
            n.val = value;
            remove(n);
            addFirst(n);
            return;
        }
        if (map.size() == cap) {
            Node lru = tail.prev;
            remove(lru);
            map.remove(lru.key);
        }
        Node n = new Node(key, value);
        map.put(key, n);
        addFirst(n);
    }
    private void remove(Node n) {
        n.prev.next = n.next;
        n.next.prev = n.prev;
    }
    private void addFirst(Node n) {
        n.next = head.next;
        n.prev = head;
        head.next.prev = n;
        head.next = n;
    }
}
5

Interview: The key insight is combining HashMap lookup and DLL recency update.

6

Visualization: put(1), put(2), get(1), put(3) => evict 2.

7

Follow-up: TTL cache, thread safety, distributed cache.

8

Complexity: O(1) get/put, O(capacity) space.

2. LFU Cache #460

Problem in brief: Implement a fixed-capacity cache that evicts the least frequently used key, breaking ties by least recent use within the same frequency.

Representative example: capacity=2; put(1,1), put(2,2), get(1); put(3,3) evicts key 2 because key 1 has higher frequency.

1

中文复述: 容量满时淘汰最低频 key; 频率相同淘汰最久未使用.

2

Brute Force: 淘汰时扫描所有 key, O(n).

3

Optimal: key->value, key->freq, freq->LinkedHashSet, minFreq.

4

Java Core:

Java
class LFUCache {
    private int capacity, minFreq = 0;
    private Map<Integer, Integer> values = new HashMap<>(), freqs = new HashMap<>();
    private Map<Integer, LinkedHashSet<Integer>> buckets = new HashMap<>();
    public LFUCache(int capacity) {
        this.capacity = capacity;
    }
    public int get(int key) {
        if (!values.containsKey(key)) {
            return -1;
        }
        increaseFreq(key);
        return values.get(key);
    }
    public void put(int key, int value) {
        if (capacity == 0) {
            return;
        }
        if (values.containsKey(key)) {
            values.put(key, value);
            increaseFreq(key);
            return;
        }
        if (values.size() == capacity) {
            int evict = buckets.get(minFreq).iterator().next();
            buckets.get(minFreq).remove(evict);
            values.remove(evict);
            freqs.remove(evict);
        }
        values.put(key, value);
        freqs.put(key, 1);
        buckets.computeIfAbsent(1, x -> new LinkedHashSet<>()).add(key);
        minFreq = 1;
    }
    private void increaseFreq(int key) {
        int f = freqs.get(key);
        buckets.get(f).remove(key);
        if (f == minFreq && buckets.get(f).isEmpty()) {
            minFreq++;
        }
        freqs.put(key, f + 1);
        buckets.computeIfAbsent(f + 1, x -> new LinkedHashSet<>()).add(key);
    }
}
5

Interview: LFU has two dimensions: frequency first, recency second.

6

Visualization: freq=1 [A,B], get(A) -> freq=1 [B], freq=2 [A].

7

Follow-up: replace LinkedHashSet with custom DLL.

8

Complexity: O(1) average.

class RandomizedSet {
    private List<Integer> list = new ArrayList<>();
    private Map<Integer, Integer> pos = new HashMap<>();
    private Random rand = new Random();
    public boolean insert(int val) {
        if (pos.containsKey(val)) {
            return false;
        }
        pos.put(val, list.size());
        list.add(val);
        return true;
    }
    public boolean remove(int val) {
        if (!pos.containsKey(val)) {
            return false;
        }
        int idx = pos.get(val), last = list.get(list.size() - 1);
        list.set(idx, last);
        pos.put(last, idx);
        list.remove(list.size() - 1);
        pos.remove(val);
        return true;
    }
    public int getRandom() {
        return list.get(rand.nextInt(list.size()));
    }
}

class RandomizedCollection {
    private List<Integer> nums = new ArrayList<>();
    private Map<Integer, Set<Integer>> idx = new HashMap<>();
    private Random rand = new Random();
    public boolean insert(int val) {
        boolean fresh = !idx.containsKey(val) || idx.get(val).isEmpty();
        idx.computeIfAbsent(val, x -> new HashSet<>()).add(nums.size());
        nums.add(val);
        return fresh;
    }
    public boolean remove(int val) {
        if (!idx.containsKey(val) || idx.get(val).isEmpty()) {
            return false;
        }
        int removeIdx = idx.get(val).iterator().next();
        int lastVal = nums.get(nums.size() - 1);
        int lastIdx = nums.size() - 1;
        idx.get(val).remove(removeIdx);
        nums.set(removeIdx, lastVal);
        idx.get(lastVal).remove(lastIdx);
        if (removeIdx < lastIdx) {
            idx.get(lastVal).add(removeIdx);
        }
        nums.remove(lastIdx);
        return true;
    }
    public int getRandom() {
        return nums.get(rand.nextInt(nums.size()));
    }
}

class MinStack {
    private Deque<int[]> stack = new ArrayDeque<>();
    public void push(int val) {
        int min = stack.isEmpty() ? val : Math.min(val, stack.peek()[1]);
        stack.push(new int[] {
            val, min
        }
        );
    }
    public void pop() {
        stack.pop();
    }
    public int top() {
        return stack.peek()[0];
    }
    public int getMin() {
        return stack.peek()[1];

6. Max Stack #716

Problem in brief: Design a stack that supports normal stack operations plus peekMax and popMax, where popMax removes the most recent maximum value.

Representative example: push(5), push(1), push(5); top()->5; popMax()->5; top()->1; peekMax()->5.

1

中文复述: stack 支持 top/pop/push/peekMax/popMax.

2

Brute Force: popMax scan O(n).

3

Optimal: DLL for stack order + TreeMap value->nodes for max lookup.

4

Java Core:

Java
class MaxStack {
    class Node {
        int val;
        Node prev, next;
        Node(int v) {
            val = v;
        }
    }
    private Node head = new Node(0), tail = new Node(0);
    private TreeMap<Integer, List<Node>> map = new TreeMap<>();
    public MaxStack() {
        head.next = tail;
        tail.prev = head;
    }
    public void push(int x) {
        Node n = new Node(x);
        add(n);
        map.computeIfAbsent(x, k -> new ArrayList<>()).add(n);
    }
    public int top() {
        return tail.prev.val;
    }
    public int pop() {
        Node n = tail.prev;
        remove(n);
        List<Node> list = map.get(n.val);
        list.remove(list.size() - 1);
        if (list.isEmpty()) {
            map.remove(n.val);
        }
        return n.val;
    }
    public int peekMax() {
        return map.lastKey();
    }
    public int popMax() {
        int max = peekMax();
        List<Node> list = map.get(max);
        Node n = list.remove(list.size() - 1);
        if (list.isEmpty()) {
            map.remove(max);
        }
        remove(n);
        return max;
    }
    private void add(Node n) {
        n.prev = tail.prev;
        n.next = tail;
        tail.prev.next = n;
        tail.prev = n;
    }
    private void remove(Node n) {
        n.prev.next = n.next;
        n.next.prev = n.prev;
    }
}
5

Interview: DLL keeps stack order, TreeMap keeps value order.

6

Visualization: stack 5->1->5, popMax removes top-most 5.

7

Follow-up: bounded values bucket optimization.

8

Complexity: O(log n) for max operations.

7. Maximum Frequency Stack #895

Problem in brief: Design a stack-like structure where pop returns the most frequent value, and ties are resolved by most recent push.

Representative example: push 5,7,5,7,4,5; pop()->5, then pop()->7 because 5 and 7 tie but 7 is more recent.

1

中文复述: pop 最高频元素, 同频返回最近 push 的.

2

Brute Force: pop scan O(n).

3

Optimal: freq[val], group[freq] stack, maxFreq.

4

Java Core:

Java
class FreqStack {
    private Map<Integer, Integer> freq = new HashMap<>();
    private Map<Integer, Deque<Integer>> group = new HashMap<>();
    private int maxFreq = 0;
    public void push(int val) {
        int f = freq.getOrDefault(val, 0) + 1;
        freq.put(val, f);
        maxFreq = Math.max(maxFreq, f);
        group.computeIfAbsent(f, x -> new ArrayDeque<>()).push(val);
    }
    public int pop() {
        int val = group.get(maxFreq).pop();
        freq.put(val, freq.get(val) - 1);
        if (group.get(maxFreq).isEmpty()) {
            maxFreq--;
        }
        return val;
    }
}

Interview: Frequency chooses bucket; stack handles recency.

6

Visualization: push 5,7,5,7,4,5; pop -> 5.

7

Follow-up: distributed heavy hitter.

8

Complexity: O(1).

8. All O`one Data Structure #432

Problem in brief: Maintain string keys with integer counts and support increment, decrement, getMaxKey, and getMinKey in constant time.

Representative example: inc("a"), inc("b"), inc("a"); getMaxKey()->"a"; getMinKey()->"b".

1

中文复述: inc/dec key, O(1) getMaxKey/getMinKey.

2

Brute Force: scan all keys O(n).

3

Optimal: DLL count buckets + key->bucket.

4

Java Core:

Java
class AllOne {
    class Bucket {
        int count;
        Set<String> keys = new HashSet<>();
        Bucket prev, next;
        Bucket(int c) {
            count = c;
        }
    }
    private Bucket head = new Bucket(0), tail = new Bucket(0);
    private Map<String, Bucket> map = new HashMap<>();
    public AllOne() {
        head.next = tail;
        tail.prev = head;
    }
    public void inc(String key) {
        if (!map.containsKey(key)) {
            Bucket b = head.next.count == 1 ? head.next : insertAfter(head, new Bucket(1));
            b.keys.add(key);
            map.put(key, b);
            return;
        }
        Bucket b = map.get(key), nb = b.next.count == b.count + 1 ? b.next : insertAfter(b, new Bucket(b.count + 1));
        move(key, b, nb);
    }
    public void dec(String key) {
        Bucket b = map.get(key);
        if (b.count == 1) {
            b.keys.remove(key);
            map.remove(key);
            if (b.keys.isEmpty()) {
                remove(b);
            }
            return;
        }
        Bucket nb = b.prev.count == b.count - 1 ? b.prev : insertAfter(b.prev, new Bucket(b.count - 1));
        move(key, b, nb);
    }
    public String getMaxKey() {
        return tail.prev == head ? "" : tail.prev.keys.iterator().next();
    }
    public String getMinKey() {
        return head.next == tail ? "" : head.next.keys.iterator().next();
    }
    private void move(String key, Bucket from, Bucket to) {
        from.keys.remove(key);
        to.keys.add(key);
        map.put(key, to);
        if (from.keys.isEmpty()) {
            remove(from);
        }
    }
    private Bucket insertAfter(Bucket p, Bucket b) {
        b.next = p.next;
        b.prev = p;
        p.next.prev = b;
        p.next = b;
        return b;
    }
    private void remove(Bucket b) {
        b.prev.next = b.next;
        b.next.prev = b.prev;
    }
}

Interview: Bucketizing counts supports local ±1 movement.

6

Visualization: count=1 [a,b], inc(a) -> count=2 [a].

7

Follow-up: remove empty buckets carefully.

8

9. Find Median from Data Stream #295

Problem in brief: Maintain a growing stream of numbers and return the median after arbitrary insertions.

Representative example: addNum(1), addNum(2), findMedian()->1.5; addNum(3), findMedian()->2.

1

中文复述: 数据流加入数字, 随时返回中位数.

2

Brute Force: each insert sort O(n log n).

3

Optimal: maxHeap lower half + minHeap upper half.

4

Java Core:

Java
class MedianFinder {
    private PriorityQueue<Integer> small = new PriorityQueue<>(Collections.reverseOrder());
    private PriorityQueue<Integer> large = new PriorityQueue<>();
    public void addNum(int num) {
        small.offer(num);
        large.offer(small.poll());
        if (large.size() > small.size()) {
            small.offer(large.poll());
        }
    }
    public double findMedian() {
        if (small.size() > large.size()) {
            return small.peek();
        }
        return (small.peek() + large.peek()) / 2.0;

class Skiplist {
    static final int MAX = 16;
    Random rand = new Random();
    class Node {
        int val;
        Node[] next = new Node[MAX];
        Node(int v) {
            val = v;
        }
    }
    private Node head = new Node(-1);
    public boolean search(int target) {
        Node cur = head;
        for (int l = MAX - 1; l >= 0; l--) while (cur.next[l] != null && cur.next[l].val < target) {
            cur = cur.next[l];
        }
        cur = cur.next[0];
        return cur != null && cur.val == target;
    }
    public void add(int num) {
        Node[] update = new Node[MAX];
        Node cur = head;
        for (int l = MAX - 1; l >= 0; l--) {
            while (cur.next[l] != null && cur.next[l].val < num) {
                cur = cur.next[l];
            }
            update[l] = cur;
        }
        Node n = new Node(num);
        for (int l = 0; l < MAX && (l == 0 || rand.nextBoolean()); l++) {
            n.next[l] = update[l].next[l];
            update[l].next[l] = n;
        }
    }
    public boolean erase(int num) {
        Node[] update = new Node[MAX];
        Node cur = head;
        for (int l = MAX - 1; l >= 0; l--) {
            while (cur.next[l] != null && cur.next[l].val < num) {
                cur = cur.next[l];
            }
            update[l] = cur;
        }
        cur = cur.next[0];
        if (cur == null || cur.val != num) {
            return false;
        }
        for (int l = 0; l < MAX && update[l].next[l] == cur; l++) {
            update[l].next[l] = cur.next[l];
        }
        return true;
    }
}

        vals = new int[height + 1][width - 'A' + 1];

        if (!formulas.containsKey(k)) {
            return vals[row][col(column)];

            r2 = rowOf(parts[1]);
            c2 = colOf(parts[1]);
        }

        return c + String.valueOf(r);

    private int rowOf(String cell) {
        return Integer.parseInt(cell.substring(1));
    }
    private int colOf(String cell) {
        return cell.charAt(0) - 'A';
    }

73. 715. Range Module

Problem in brief: Maintain covered numeric intervals under addRange, removeRange, and queryRange.

Representative example: add [10,20); query [10,14)->true; remove [14,16); query [13,15)->false.

Approach: TreeMap disjoint intervals; addRange merges all overlapping intervals, queryRange checks floor interval containment, removeRange splits/trim overlaps. Complexity: O(k log n).

Java Core:

Java
class RangeModule {
    private TreeMap<Integer, Integer> map = new TreeMap<>();
    public void addRange(int left, int right) {
        Integer start = map.floorKey(left);
        if (start != null && map.get(start) >= left) {
            left = start;
            right = Math.max(right, map.remove(start));
        }
        for (start = map.ceilingKey(left); start != null && start <= right; start = map.ceilingKey(left)) {
            right = Math.max(right, map.remove(start));
        }
        map.put(left, right);
    }
    public boolean queryRange(int left, int right) {
        Integer start = map.floorKey(left);
        return start != null && map.get(start) >= right;
    }
    public void removeRange(int left, int right) {
        Integer start = map.floorKey(left);
        if (start != null && map.get(start) > left) {
            int end = map.get(start);
            if (end > right) {
                map.put(right, end);
            }
            map.put(start, left);
        }
        for (start = map.ceilingKey(left); start != null && start < right; start = map.ceilingKey(left)) {
            int end = map.remove(start);
            if (end > right) {
                map.put(right, end);
                break;
            }
        }
    }
}

80. 1157. Online Majority Element In Subarray

Problem in brief: Preprocess an array and answer whether a value appears at least threshold times in a subarray.

Representative example: arr=[1,1,2,2,1,1]; query(0,5,4)->1; query(0,3,3)->-1.

Approach: segment tree stores Boyer-Moore candidate per range; query candidate, then count its occurrences using value -> sorted indices and binary search to verify threshold. Complexity: O(log n) candidate + O(log n) count.

Java Core:

Java
class MajorityChecker {
    class Node {
        int val, cnt;
        Node(int v, int c) {
            val = v;
            cnt = c;
        }
    }
    private int[] arr;
    private Node[] tree;
    private Map<Integer, List<Integer>> pos = new HashMap<>();
    public MajorityChecker(int[] arr) {
        this.arr = arr;
        tree = new Node[arr.length * 4];
        for (int i = 0; i < arr.length; i++) {
            pos.computeIfAbsent(arr[i], k -> new ArrayList<>()).add(i);
        }
        build(1, 0, arr.length - 1);
    }
    public int query(int left, int right, int threshold) {
        int cand = query(1, 0, arr.length - 1, left, right).val;
        List<Integer> list = pos.get(cand);
        int count = upper(list, right) - lower(list, left);
        return count >= threshold ? cand : -1;
    }
    private Node merge(Node a, Node b) {
        if (a.val == b.val) {
            return new Node(a.val, a.cnt + b.cnt);
        }
        return a.cnt > b.cnt ? new Node(a.val, a.cnt - b.cnt) : new Node(b.val, b.cnt - a.cnt);
    }
    private void build(int idx, int l, int r) {
        if (l == r) {
            tree[idx] = new Node(arr[l], 1);
            return;
        }
        int m = (l + r) / 2;
        build(idx * 2, l, m);
        build(idx * 2 + 1, m + 1, r);
        tree[idx] = merge(tree[idx * 2], tree[idx * 2 + 1]);
    }
    private Node query(int idx, int l, int r, int ql, int qr) {
        if (ql <= l && r <= qr) {
            return tree[idx];
        }
        int m = (l + r) / 2;
        if (qr <= m) {
            return query(idx * 2, l, m, ql, qr);
        }
        if (ql > m) {
            return query(idx * 2 + 1, m + 1, r, ql, qr);
        }
        return merge(query(idx * 2, l, m, ql, qr), query(idx * 2 + 1, m + 1, r, ql, qr));
    }
    private int lower(List<Integer> a, int x) {
        int l = 0, r = a.size();
        while (l < r) {
            int m = (l + r) / 2;
            if (a.get(m) < x) {
                l = m + 1;
            } else r = m;
        }
        return l;
    }
    private int upper(List<Integer> a, int x) {
        int l = 0, r = a.size();
        while (l < r) {
            int m = (l + r) / 2;
            if (a.get(m) <= x) {
                l = m + 1;
            } else r = m;
        }
        return l;
    }
}

Problem in brief: Maintain many stacks of fixed capacity, pushing to the leftmost non-full stack and popping from the rightmost non-empty stack or a chosen stack.

Representative example: capacity=2; pushes fill stack 0 then 1; popAtStack(0) creates a hole used by the next push.

Approach: list of stacks, min-heap of indices with free capacity, and right pointer/cleanup for non-empty stacks; discard stale heap indices before use. Complexity: push/popAtStack O(log n), pop amortized O(log n).

Java Core:

Java
class DinnerPlates {
    private int capacity;
    private List<Deque<Integer>> stacks = new ArrayList<>();
    private TreeSet<Integer> notFull = new TreeSet<>();
    public DinnerPlates(int capacity) {
        this.capacity = capacity;
    }
    public void push(int val) {
        if (notFull.isEmpty()) {
            stacks.add(new ArrayDeque<>());
            notFull.add(stacks.size() - 1);
        }
        int i = notFull.first();
        stacks.get(i).push(val);
        if (stacks.get(i).size() == capacity) {
            notFull.remove(i);
        }
    }
    public int pop() {
        cleanTail();
        return stacks.isEmpty() ? -1 : popAtStack(stacks.size() - 1);
    }
    public int popAtStack(int index) {
        if (index < 0 || index >= stacks.size() || stacks.get(index).isEmpty()) {
            return -1;
        }
        int val = stacks.get(index).pop();
        notFull.add(index);
        cleanTail();
        return val;
    }
    private void cleanTail() {
        while (!stacks.isEmpty() && stacks.get(stacks.size() - 1).isEmpty()) {
            notFull.remove(stacks.size() - 1);
            stacks.remove(stacks.size() - 1);
        }
    }
}

Approach: build MST/forest with Kruskal because connectivity under max-edge limit is preserved; answer query by checking max edge on path via binary-lifting LCA and comparing with limit. Complexity: build O(E log E + n log n), query O(log n).

Java Core:

Java
class DistanceLimitedPathsExist {
    class DSU {
        int[] p;
        DSU(int n) {
            p = new int[n];
            for (int i = 0; i < n; i++) {
                p[i] = i;
            }
        }
        int find(int x) {
            return p[x] == x ? x : (p[x] = find(p[x]));
        }
        boolean union(int a, int b) {
            a = find(a);
            b = find(b);
            if (a == b) {
                return false;
            }
            p[a] = b;
            return true;
        }
    }
    private int LOG;
    private int[][] up, mx;
    private int[] depth, comp;
    private List<int[]>[] graph;
    public DistanceLimitedPathsExist(int n, int[][] edgeList) {
        Arrays.sort(edgeList, Comparator.comparingInt(e -> e[2]));
        graph = new ArrayList[n];
        for (int i = 0; i < n; i++) {
            graph[i] = new ArrayList<>();
        }
        DSU dsu = new DSU(n);
        for (int[] e : edgeList) if (dsu.union(e[0], e[1])) {
            graph[e[0]].add(new int[] {
                e[1], e[2]
            }
            );
            graph[e[1]].add(new int[] {
                e[0], e[2]
            }
            );
        }
        LOG = 1;
        while ((1 < < LOG) <= n) {
            LOG++;
        }
        up = new int[LOG][n];
        mx = new int[LOG][n];
        depth = new int[n];
        comp = new int[n];
        Arrays.fill(comp, -1);
        for (int i = 0; i < n; i++) if (comp[i] == -1) {
            dfs(i, i, 0, i, 0);
        }
        for (int j = 1; j < LOG; j++) for (int i = 0; i < n; i++) {
            up[j][i] = up[j - 1][up[j - 1][i]];
            mx[j][i] = Math.max(mx[j - 1][i], mx[j - 1][up[j - 1][i]]);
        }
    }
    public boolean query(int p, int q, int limit) {
        if (comp[p] != comp[q]) {
            return false;
        }
        return maxEdge(p, q) < limit;
    }
    private void dfs(int u, int parent, int w, int cid, int dep) {
        comp[u] = cid;
        up[0][u] = parent;
        mx[0][u] = w;
        depth[u] = dep;
        for (int[] e : graph[u]) if (e[0] != parent) {
            dfs(e[0], u, e[1], cid, dep + 1);
        }
    }
    private int maxEdge(int a, int b) {
        int ans = 0;
        if (depth[a] < depth[b]) {
            int t = a;
            a = b;
            b = t;
        }
        int diff = depth[a] - depth[b];
        for (int j = 0; diff > 0; j++, diff >>= 1) if ((diff & 1) == 1) {
            ans = Math.max(ans, mx[j][a]);
            a = up[j][a];
        }
        if (a == b) {
            return ans;
        }
        for (int j = LOG - 1; j >= 0; j--) if (up[j][a] != up[j][b]) {
            ans = Math.max(ans, Math.max(mx[j][a], mx[j][b]));
            a = up[j][a];
            b = up[j][b];
        }
        return Math.max(ans, Math.max(mx[0][a], mx[0][b]));
    }
}

102. 1825. Finding MK Average

Problem in brief: Maintain the last m numbers and compute the average after removing the smallest k and largest k values.

Representative example: m=3,k=1; after [3,1,10], MKAverage removes 1 and 10 and returns 3.

Approach: queue last m elements plus three ordered multisets: low k, middle m-2k with sum, high k; rebalance after add/evict. Complexity: O(log m), get O(1).

Java Core:

Java
class MKAverage {
    private int m, k, lowSize = 0, midSize = 0, highSize = 0;
    private long midSum = 0;
    private Queue<Integer> q = new ArrayDeque<>();
    private TreeMap<Integer, Integer> low = new TreeMap<>(), mid = new TreeMap<>(), high = new TreeMap<>();
    public MKAverage(int m, int k) {
        this.m = m;
        this.k = k;
    }
    public void addElement(int num) {
        q.offer(num);
        if (!low.isEmpty() && num <= low.lastKey()) {
            addLow(num);
        } else if (!high.isEmpty() && num >= high.firstKey()) addHigh(num);
        else addMid(num);
        rebalance();
        if (q.size() > m) {
            remove(q.poll());
        }
    }
    public int calculateMKAverage() {
        return q.size() < m ? -1 : (int)(midSum / (m - 2 * k));
    }
    private void remove(int x) {
        if (dec(low, x)) {
            lowSize--;
        } else if (dec(high, x)) highSize--;
        else {
            dec(mid, x);
            midSize--;
            midSum -= x;
        }
        rebalance();
    }
    private void rebalance() {
        while (lowSize > k) {
            int x = low.lastKey();
            dec(low, x);
            lowSize--;
            addMid(x);
        }
        while (highSize > k) {
            int x = high.firstKey();
            dec(high, x);
            highSize--;
            addMid(x);
        }
        while (lowSize < k && midSize > 0) {
            int x = mid.firstKey();
            removeMid(x);
            addLow(x);
        }
        while (highSize < k && midSize > 0) {
            int x = mid.lastKey();
            removeMid(x);
            addHigh(x);
        }
    }
    private void addLow(int x) {
        inc(low, x);
        lowSize++;
    }
    private void addMid(int x) {
        inc(mid, x);
        midSize++;
        midSum += x;
    }
    private void addHigh(int x) {
        inc(high, x);
        highSize++;
    }
    private void removeMid(int x) {
        dec(mid, x);
        midSize--;
        midSum -= x;
    }
    private void inc(TreeMap<Integer, Integer> t, int x) {
        t.merge(x, 1, Integer::sum);
    }
    private boolean dec(TreeMap<Integer, Integer> t, int x) {
        Integer c = t.get(x);
        if (c == null) {
            return false;
        }
        if (c == 1) {
            t.remove(x);
        } else t.put(x, c - 1);
        return true;
    }
}

        children = new ArrayList[parent.length];

        if (locked[num] != user) {
            return false;
        }

        if (locked[num] != 0 || hasLockedAncestor(num) || !unlockDesc(num)) {
            return false;
        }
        locked[num] = user;
        return true;

        pull(idx);

    private int find(int idx, int l, int r, int maxRow, int k) {
        if (l > maxRow || max[idx] < k) {
            return -1;
        }
        if (l == r) {
            return l;
        }
        int mid = (l + r) / 2;
        int left = find(idx * 2, l, mid, maxRow, k);
        return left != -1 ? left : find(idx * 2 + 1, mid + 1, r, maxRow, k);
    }
    private long query(int idx, int l, int r, int ql, int qr) {
        if (qr < l || r < ql) {
            return 0;
        }
        if (ql <= l && r <= qr) {
            return sum[idx];
        }
        int mid = (l + r) / 2;
        return query(idx * 2, l, mid, ql, qr) + query(idx * 2 + 1, mid + 1, r, ql, qr);
    }
    private void pull(int idx) {
        max[idx] = Math.max(max[idx * 2], max[idx * 2 + 1]);
        sum[idx] = sum[idx * 2] + sum[idx * 2 + 1];
    }
}

class Graph {
    private List<int[]>[] g;
    public Graph(int n, int[][] edges) {
        g = new ArrayList[n];
        for (int i = 0; i < n; i++) {
            g[i] = new ArrayList<>();
        }
        for (int[] e : edges) {
            addEdge(e);
        }
    }
    public void addEdge(int[] edge) {
        g[edge[0]].add(new int[] {
            edge[1], edge[2]
        }
        );
    }
    public int shortestPath(int node1, int node2) {
        int n = g.length;
        long[] dist = new long[n];
        Arrays.fill(dist, Long.MAX_VALUE / 4);
        dist[node1] = 0;
        PriorityQueue<long[]> pq = new PriorityQueue<>(Comparator.comparingLong(a -> a[1]));
        pq.offer(new long[] {
            node1, 0
        }
        );
        while (!pq.isEmpty()) {
            long[] cur = pq.poll();
            int u = (int)cur[0];
            if (cur[1] != dist[u]) {
                continue;
            }
            if (u == node2) {
                return (int)cur[1];
            }
            for (int[] e : g[u]) if (dist[e[0]] > cur[1] + e[1]) {
                dist[e[0]] = cur[1] + e[1];
                pq.offer(new long[] {
                    e[0], dist[e[0]]
                }
                );
            }
        }
        return -1;
    }
}

Approach: insertion-order queue, running sum for mean, two ordered multisets for median (upper middle on even size), and frequency map plus ordered set for mode (highest frequency, smallest value tie). Complexity: O(log n) update, O(1) query endpoints.

Java Core:

Java
class StatisticsTracker {
    private Queue<Integer> q = new ArrayDeque<>();
    private long sum = 0;
    private TreeMap<Integer, Integer> low = new TreeMap<>(), high = new TreeMap<>();
    private int lowSize = 0, highSize = 0;
    private Map<Integer, Integer> freq = new HashMap<>();
    private TreeSet<int[]> modes = new TreeSet<>((a, b) -> a[1] != b[1] ? Integer.compare(b[1], a[1]) : Integer.compare(a[0], b[0]));
    public void addNumber(int number) {
        q.offer(number);
        sum += number;
        addMedian(number);
        changeFreq(number, 1);
    }
    public void removeFirstAddedNumber() {
        int x = q.poll();
        sum -= x;
        removeMedian(x);
        changeFreq(x, -1);
    }
    public int getMean() {
        return (int)(sum / q.size());
    }
    public int getMedian() {
        return high.firstKey();
    }
    public int getMode() {
        return modes.first()[0];
    }
    private void addMedian(int x) {
        if (high.isEmpty() || x >= high.firstKey()) {
            inc(high, x);
            highSize++;
        } else {
            inc(low, x);
            lowSize++;
        }
        balance();
    }
    private void removeMedian(int x) {
        if (dec(high, x)) {
            highSize--;
        } else {
            dec(low, x);
            lowSize--;
        }
        balance();
    }
    private void balance() {
        while (highSize < lowSize) {
            int x = low.lastKey();
            dec(low, x);
            lowSize--;
            inc(high, x);
            highSize++;
        }
        while (highSize > lowSize + 1) {
            int x = high.firstKey();
            dec(high, x);
            highSize--;
            inc(low, x);
            lowSize++;
        }
    }
    private void changeFreq(int x, int delta) {
        int old = freq.getOrDefault(x, 0);
        if (old > 0) modes.remove(new int[] {
            x, old
        }
        );
        int now = old + delta;
        if (now == 0) {
            freq.remove(x);
        } else {
            freq.put(x, now);
            modes.add(new int[] {
                x, now
            }
            );
        }
    }
    private void inc(TreeMap<Integer, Integer> t, int x) {
        t.merge(x, 1, Integer::sum);
    }
    private boolean dec(TreeMap<Integer, Integer> t, int x) {
        Integer c = t.get(x);
        if (c == null) {
            return false;
        }
        if (c == 1) {
            t.remove(x);
        } else t.put(x, c - 1);
        return true;
    }
}

    private int limit;
    private Queue<int[]> q = new ArrayDeque<>();
    private Set<String> seen = new HashSet<>();
    private Map<Integer, List<Integer>> times = new HashMap<>();
    private Map<Integer, Integer> removed = new HashMap<>();
    public Router(int memoryLimit) {
        limit = memoryLimit;
    }
    public boolean addPacket(int source, int destination, int timestamp) {
        String key = key(source, destination, timestamp);
        if (seen.contains(key)) {
            return false;
        }
        if (q.size() == limit) {
            evict();
        }
        int[] p = {
            source, destination, timestamp
        }
        ;
        q.offer(p);
        seen.add(key);
        times.computeIfAbsent(destination, k -> new ArrayList<>()).add(timestamp);
        return true;
    }

        return q.isEmpty() ? new int[0] : evict();
    }

        int start = removed.getOrDefault(destination, 0);
        return upper(a, start, endTime) - lower(a, start, startTime);
    }
    private int[] evict() {
        int[] p = q.poll();
        seen.remove(key(p[0], p[1], p[2]));
        removed.merge(p[1], 1, Integer::sum);
        return p;
    }
    private String key(int s, int d, int t) {
        return s + "#" + d + "#" + t;
    }
    private int lower(List<Integer> a, int lo, int x) {
        int hi = a.size();
        while (lo < hi) {
            int m = (lo + hi) / 2;
            if (a.get(m) < x) {
                lo = m + 1;
            } else hi = m;
        }
        return lo;
    }
    private int upper(List<Integer> a, int lo, int x) {
        int hi = a.size();
        while (lo < hi) {
            int m = (lo + hi) / 2;
            if (a.get(m) <= x) {
                lo = m + 1;
            } else hi = m;
        }

    }

Approach: (user,item)->bid map plus item->TreeSet ordered by amount desc then userId desc. LeetCode uses addBid, updateBid, removeBid, and getHighestBidder; add/update move the bidder entry in that item's ordered set. Complexity: O(log bidders).

class AuctionSystem {
    class Bid {
        int userId, itemId, amount;
        Bid(int u, int i, int a) {
            userId = u;
            itemId = i;
            amount = a;
        }
    }
    private Map<String, Bid> bids = new HashMap<>();
    private Map<Integer, TreeSet<Bid>> byItem = new HashMap<>();
    private Comparator<Bid> cmp = (a, b) -> a.amount != b.amount ? b.amount - a.amount : b.userId - a.userId;
    public void addBid(int userId, int itemId, int bidAmount) {
        putBid(userId, itemId, bidAmount);
    }
    public void updateBid(int userId, int itemId, int newAmount) {
        putBid(userId, itemId, newAmount);
    }
    private void putBid(int userId, int itemId, int amount) {
        String key = userId + "#" + itemId;
        Bid old = bids.get(key);
        if (old != null) {
            byItem.get(itemId).remove(old);
        }
        Bid bid = new Bid(userId, itemId, amount);
        bids.put(key, bid);
        byItem.computeIfAbsent(itemId, k -> new TreeSet<>(cmp)).add(bid);
    }
    public void removeBid(int userId, int itemId) {
        Bid old = bids.remove(userId + "#" + itemId);
        if (old != null) {
            byItem.get(itemId).remove(old);
        }
    }
    public int getHighestBidder(int itemId) {
        TreeSet<Bid> set = byItem.get(itemId);
        return set == null || set.isEmpty() ? -1 : set.first().userId;
    }
}

Approach: orderId->(type,price) plus bucket map (type,price)->set(orderIds). LeetCode uses modifyOrder(orderId,newPrice) and getOrdersAtPrice(orderType,price); modifying keeps the old type and moves only the price bucket. Complexity: update O(1) average, query O(k).

class OrderManagementSystem {
    class Order {
        int id, price;
        String type;
        Order(int i, String t, int p) {
            id = i;
            type = t;
            price = p;
        }
    }
    private Map<Integer, Order> orders = new HashMap<>();
    private Map<String, Map<Integer, Set<Integer>>> bucket = new HashMap<>();
    public void addOrder(int orderId, String orderType, int price) {
        Order o = new Order(orderId, orderType, price);
        orders.put(orderId, o);
        bucket.computeIfAbsent(orderType, k -> new HashMap<>()).computeIfAbsent(price, k -> new HashSet<>()).add(orderId);
    }
    public void modifyOrder(int orderId, int newPrice) {
        Order o = orders.get(orderId);
        bucket.get(o.type).get(o.price).remove(orderId);
        o.price = newPrice;
        bucket.computeIfAbsent(o.type, k -> new HashMap<>()).computeIfAbsent(newPrice, k -> new HashSet<>()).add(orderId);
    }
    public void cancelOrder(int orderId) {
        Order o = orders.remove(orderId);
        if (o == null) {
            return;
        }
        bucket.get(o.type).get(o.price).remove(orderId);
    }
    public int[] getOrdersAtPrice(String orderType, int price) {
        Set<Integer> set = bucket.getOrDefault(orderType, Map.of()).getOrDefault(price, Set.of());
        int[] ans = new int[set.size()];
        int i = 0;
        for (int id : set) {
            ans[i++] = id;
        }
        return ans;
    }
}

133. 3829. Design Ride Sharing System

Problem in brief: Manage ride requests, drivers, matches, and trip state transitions.

Representative example: Request a ride, assign an available driver, complete or cancel the ride, and update future availability.

Approach: FIFO queues preserve rider and driver arrival order; cancellation marks a pending rider inactive and matching lazily skips canceled riders. LeetCode uses matchDriverWithRider() and returns [-1,-1] when either side is unavailable. Complexity: O(1) amortized.

Java Core:

Java
class RideSharingSystem {
    private Queue<Integer> riders = new ArrayDeque<>(), drivers = new ArrayDeque<>();
    private Set<Integer> activeRiders = new HashSet<>();
    public RideSharingSystem() {
    }
    public void addRider(int riderId) {
        riders.offer(riderId);
        activeRiders.add(riderId);
    }
    public void addDriver(int driverId) {
        drivers.offer(driverId);
    }
    public int[] matchDriverWithRider() {
        while (!riders.isEmpty() && !activeRiders.contains(riders.peek())) {
            riders.poll();
        }
        if (riders.isEmpty() || drivers.isEmpty()) return new int[] {
            -1, -1
        }
        ;
        int driver = drivers.poll(), rider = riders.poll();
        activeRiders.remove(rider);
        return new int[] {
            driver, rider
        }
        ;
    }
    public void cancelRider(int riderId) {
        activeRiders.remove(riderId);
    }
}

134. 3885. Design Event Manager

Problem in brief: Maintain events and support creating, canceling, updating, or querying events with ordering constraints.

Representative example: Create events with times; query upcoming events returns active events in time order; canceled events disappear lazily or eagerly.

Approach: eventId->priority plus TreeSet ordered by priority desc then eventId asc; update moves event to new priority, pollHighest removes and returns the top event, or -1 if empty. Complexity: O(log n).

Java Core:

Java
class EventManager {
    private Map<Integer, Integer> priority = new HashMap<>();
    private TreeSet<int[]> set = new TreeSet<>((a, b) -> a[1] != b[1] ? b[1] - a[1] : a[0] - b[0]);
    public EventManager(int[][] events) {
        for (int[] e : events) {
            priority.put(e[0], e[1]);
            set.add(new int[] {
                e[0], e[1]
            }
            );
        }
    }
    public void updatePriority(int eventId, int newPriority) {
        Integer old = priority.get(eventId);
        if (old == null) {
            return;
        }
        set.remove(new int[] {
            eventId, old
        }
        );
        priority.put(eventId, newPriority);
        set.add(new int[] {
            eventId, newPriority
        }
        );
    }
    public int pollHighest() {
        if (set.isEmpty()) {
            return -1;
        }
        int[] e = set.pollFirst();
        priority.remove(e[0]);
        return e[0];
    }
}
