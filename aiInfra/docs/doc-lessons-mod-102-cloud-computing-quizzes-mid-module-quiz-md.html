Module 02: 云计算（Cloud Computing） - Mid-Module Quiz

Time Limit: 25 minutes Passing Score: 80% (16/20 questions) Coverage: Lessons 01-04 (Cloud Architecture, AWS 云（AWS）, GCP 云（GCP）, Azure 云（Azure）)

Section 1: Cloud Architecture for ML (5 questions)

Q1. What is the primary advantage of cloud-native ML architectures?

a) Lower total cost than on-premises b) Elasticity and scalability on demand c) Faster training times guaranteed d) No need for monitoring

Answer: B

Q2. Which storage type is best for training datasets in the cloud?

a) Block storage (EBS) b) Object storage (S3, GCS) c) Relational database d) Local SSD only

Answer: B

Q3. What is the purpose of a VPC in cloud ML infrastructure?

a) To reduce costs b) To provide isolated network environment c) To speed up training d) To store models

Answer: B

Q4. Which compute option is most cost-effective for batch training jobs?

a) On-demand instances b) Reserved instances c) Spot/preemptible instances d) Dedicated hosts

Answer: C

Q5. What is the trade-off of using serverless for ML inference?

a) Higher latency vs lower operational overhead b) Lower cost vs higher complexity c) Better security vs worse performance d) No trade-offs exist

Answer: A

Section 2: AWS 云（AWS） for 机器学习基础设施（ML Infrastructure） (5 questions)

Q6. Which AWS 云（AWS） service provides managed Kubernetes 容器编排（Kubernetes）?

a) ECS b) EKS 托管 Kubernetes（EKS） c) Fargate d) Lambda

Answer: B

Q7. What is the primary use of AWS 云（AWS） SageMaker 托管 ML（SageMaker）?

a) Object storage b) Managed ML training and deployment c) Container orchestration d) Networking

Answer: B

Q8. Which AWS 云（AWS） storage service is best for large-scale object storage?

a) EBS b) EFS c) S3 d) Instance Store

Answer: C

Q9. What does IAM stand for in AWS 云（AWS）?

a) Internet Access Management b) Identity and Access Management c) Infrastructure Automation Manager d) Image Asset Manager

Answer: B

Q10. Which EC2 instance type is optimized for GPU 加速（GPU） workloads?

a) t3.large b) m5.xlarge c) p3.2xlarge d) c5.4xlarge

Answer: C

Section 3: Google Cloud Platform for ML (5 questions)

Q11. What is Google's managed ML platform called?

a) AI Platform b) Vertex AI 托管 ML（Vertex AI） c) Cloud ML Engine d) ML Studio

Answer: B

Q12. Which GCP 云（GCP） service provides managed Kubernetes 容器编排（Kubernetes）?

a) GCE b) GAE c) GKE 托管 Kubernetes（GKE） d) Cloud Run

Answer: C

Q13. What is a TPU in Google Cloud?

a) Tensor Processing Unit for ML acceleration b) Total Performance Upgrade c) 训练（Training） Pipeline Utility d) Temporary Processing Unit

Answer: A

Q14. Which GCP 云（GCP） storage is equivalent to AWS 云（AWS） S3?

a) Persistent Disk b) Cloud Storage c) Filestore d) Cloud SQL

Answer: B

Q15. What is the advantage of TPUs over GPUs for certain ML workloads?

a) Cheaper always b) Optimized for TensorFlow 深度学习框架（TensorFlow） operations c) Better for gaming d) No advantages

Answer: B

Section 4: Azure 云（Azure） for 机器学习基础设施（ML Infrastructure） (5 questions)

Q16. What is Azure 云（Azure）'s managed ML service called?

a) Azure 云（Azure） AI b) Azure 云（Azure） 机器学习（Machine Learning） c) Azure 云（Azure） ML Studio d) Azure 云（Azure） Cognitive Services

Answer: B

Q17. Which Azure 云（Azure） service provides managed Kubernetes 容器编排（Kubernetes）?

a) Azure 云（Azure） Container Instances b) Azure 云（Azure） Functions c) AKS 托管 Kubernetes（AKS） (Azure Kubernetes 容器编排（Kubernetes） Service) d) Azure 云（Azure） App Service

Answer: C

Q18. What unique capability does Azure 云（Azure） OpenAI Service provide?

a) Free GPU 加速（GPU） access b) Access to GPT models through Azure 云（Azure） c) Automated model training d) Unlimited storage

Answer: B

Q19. Which Azure 云（Azure） storage is equivalent to AWS 云（AWS） S3?

a) Azure 云（Azure） Files b) Azure 云（Azure） Disk c) Blob Storage d) Table Storage

Answer: C

Q20. What is a key advantage of Azure 云（Azure） for enterprise customers?

a) Lowest cost b) Integration with Microsoft ecosystem c) Fastest performance d) No security features

Answer: B
